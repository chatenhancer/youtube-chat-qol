安装 Chat Enhancer

从 Chrome Web Store 或 Firefox Add-ons 获取，或从源码构建扩展用于本地开发。Chrome 版本通常也可在其他基于 Chromium 的浏览器中使用。

开始使用

安装后，如果 YouTube 已经打开，请刷新一次，然后在直播聊天所在的位置使用扩展。

  1. 从浏览器商店安装 使用上方的 Chrome 或 Firefox 按钮。其他基于 Chromium 的浏览器通常可以使用 Chrome 构建版本。
  2. 打开或刷新 YouTube 直播聊天 刷新已经打开的标签页，让扩展连接到实时聊天框架。当收件箱图标 出现在 YouTube 聊天中时，就表示扩展正在工作。
  3. 需要时调整完整设置 扩展会在实时聊天中自动运行。要查看完整设置，请点击浏览器工具栏中的扩展图标 , 然后选择 Chat Enhancer 以显示所有选项。如果想更快访问，可以将它固定。
  4. 使用聊天增强功能 消息菜单、头像卡片、收件箱按钮、常用表情和 Tab 命令会显示在 YouTube 聊天界面中，并可直接使用。

隐私

以本地为先的聊天工具

扩展旨在将日常聊天增强保留在你的设备上。翻译是唯一需要外部服务的功能，并且只会在你选择目标语言后运行。

阅读完整隐私政策
范围
仅在 YouTube 直播聊天和直播聊天回放页面运行。
存储
设置、收件箱消息、监视关键词和常用表情计数都会保存在浏览器扩展存储中。
翻译
默认关闭。启用后，消息文本会发送到 Google Translate，但不会附带你的 YouTube Cookie 或凭据。
跟踪
没有分析、浏览历史收集、数据转售或扩展自有服务器。

聊天命令用 Tab 展开

在 YouTube 聊天输入框中输入命令，然后按 Tab。命令不会自动发送，未知的斜杠命令会保持不变。使用 // 可发送字面意义的斜杠命令。

行内命令： /mention、/reply、/time 和 /timeuntil 可以在句子中展开。

整条输入命令： /quote、/again、/repeat、/help 以及所有 /set... 命令应作为输入框中的唯一文本。

文本命令

/mention or /reply
插入对最新收件箱消息作者的提及。
/quote
引用收件箱中保存的最新消息。
/again or /repeat
恢复你上一次发送的消息，在可用时包含富表情内容。
/time utc
插入支持的时区或城市的当前时间。
支持的别名： utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/timeuntil 7:45pm
插入距离下一个匹配本地时间的剩余时间。
接受的格式： 7, 7am, 7 am, 7:45am, 7:45 am, 19, 19:00, 19:00:30, and 7:45:30 pm.
/help
在 YouTube 聊天中打开命令帮助卡片。

设置命令

/settranslateto english or /settranslateto off
设置翻译目标语言。使用 off 关闭翻译。
/settranslationdisplay replace or /settranslationdisplay below
用翻译替换原始消息。使用 below 在下方显示翻译。
/setquotelength 120
将引用长度设置为 80、120、180 或 240。
/setsound on or /setsound off
开启或关闭收件箱提示音。
/setopenchannelsinpopup on or /setopenchannelsinpopup off
在弹出窗口中打开频道。使用 off 正常打开。

Chat Enhancer for YouTube 与 YouTube 或 Google 没有关联。
