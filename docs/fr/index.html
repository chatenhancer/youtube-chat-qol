Installer Chat Enhancer

Installez-la depuis le Chrome Web Store ou Firefox Add-ons, ou compilez l'extension depuis le code source pour le développement local. La version Chrome peut aussi fonctionner dans d'autres navigateurs basés sur Chromium.

Premiers pas

Installez, actualisez YouTube une fois s'il était déjà ouvert, puis utilisez l'extension là où le chat en direct se trouve déjà.

  1. Installer depuis la boutique de votre navigateur Utilisez le bouton Chrome ou Firefox ci-dessus. Les autres navigateurs basés sur Chromium peuvent généralement utiliser la version Chrome.
  2. Ouvrir ou actualiser le chat en direct YouTube Actualisez les onglets déjà ouverts pour que l’extension puisse se connecter au cadre du chat en direct. Vous saurez qu’elle fonctionne lorsque l’icône de boîte de réception apparaît dans le chat YouTube.
  3. Ajuster les réglages complets si nécessaire L’extension fonctionne automatiquement dans le chat en direct. Pour accéder à tous les réglages, cliquez sur l’icône des extensions dans la barre d’outils du navigateur , puis choisissez Chat Enhancer pour afficher toutes les options. Épinglez-la si vous voulez un accès plus rapide.
  4. Utiliser les améliorations du chat Les menus de messages, cartes d’avatar, bouton de boîte de réception, emojis fréquents et commandes Tab apparaissent et peuvent être utilisés dans l’interface de chat YouTube.

Confidentialité

Outils de chat locaux d'abord

L'extension est conçue pour garder les améliorations courantes du chat sur votre appareil. La traduction est la seule fonction qui nécessite un service externe, et elle ne s'exécute qu'après le choix d'une langue cible.

Lire la politique de confidentialité complète
Portée
Fonctionne uniquement sur les pages de chat en direct YouTube et de replay du chat en direct.
Stockage
Les paramètres, messages de la boîte de réception, mots-clés surveillés et compteurs d’emojis fréquents restent dans le stockage de l’extension du navigateur.
Traduction
Désactivée par défaut. Si elle est activée, le texte des messages est envoyé à Google Translate sans vos cookies ni identifiants YouTube.
Suivi
Pas d'analytics, de collecte d'historique de navigation, de revente de données ni de serveur appartenant à l'extension.

Référence

Les commandes du chat se développent avec Tab

Tapez une commande dans le champ du chat YouTube, puis appuyez sur Tab. Les commandes ne sont pas envoyées automatiquement et les commandes slash inconnues sont laissées telles quelles. Utilisez // pour envoyer une commande slash littérale.

Commandes en ligne : /mention, /reply, /time et /timeuntil peuvent se développer dans une phrase.

Commandes en saisie complète : /quote, /again, /repeat, /help et toutes les commandes /set... doivent être le seul texte dans le champ.

Commandes de texte

/mention or /reply
Insère une mention pour l’auteur du dernier message de votre boîte de réception.
/quote
Cite le dernier message enregistré dans votre boîte de réception.
/again or /repeat
Restaure votre dernier message envoyé, y compris le contenu emoji enrichi quand il est disponible.
/time utc
Insère l'heure actuelle pour un fuseau horaire ou une ville compatible.
Alias pris en charge : utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/timeuntil 7:45pm
Insère le temps restant jusqu'à la prochaine heure locale correspondante.
Formats acceptés : 7, 7am, 7 am, 7:45am, 7:45 am, 19, 19:00, 19:00:30, and 7:45:30 pm.
/help
Ouvre la carte d'aide des commandes dans le chat YouTube.

Commandes de réglages

/settranslateto english or /settranslateto off
Définit la langue cible de traduction. Utilisez off pour désactiver la traduction.
/settranslationdisplay replace or /settranslationdisplay below
Remplace les messages originaux par les traductions. Utilisez below pour afficher les traductions dessous.
/setquotelength 120
Définit la longueur de citation sur 80, 120, 180 ou 240.
/setsound on or /setsound off
Active ou désactive les sons de la boîte de réception.
/setopenchannelsinpopup on or /setopenchannelsinpopup off
Ouvre les chaînes dans des fenêtres popup. Utilisez off pour les ouvrir normalement.
