เริ่มต้นใช้งาน

ติดตั้ง รีเฟรช YouTube หนึ่งครั้งถ้าเปิดอยู่แล้ว จากนั้นใช้ส่วนขยายในพื้นที่ live chat ได้เลย

  1. ติดตั้งจาก store ของเบราว์เซอร์ ใช้ปุ่ม Chrome หรือ Firefox ด้านบน เบราว์เซอร์ Chromium อื่นโดยทั่วไปใช้ build ของ Chrome ได้
  2. เปิดหรือรีเฟรช YouTube live chat รีเฟรชแท็บ YouTube live chat ที่เปิดอยู่ก่อนแล้ว เอฟเฟกต์เริ่มต้นสั้น ๆ อาจเล่นขึ้นมา และเมื่อไอคอนกล่องข้อความ ปรากฏในแชท YouTube แปลว่าฟีเจอร์เสริมของแชทพร้อมใช้งานแล้ว
  3. ปรับการตั้งค่าเต็มเมื่อจำเป็น ส่วนขยายทำงานอัตโนมัติใน live chat หากต้องการตั้งค่าเต็ม ให้คลิกไอคอนส่วนขยายบนแถบเครื่องมือเบราว์เซอร์ , แล้วเลือก Chat Enhancer เพื่อดูตัวเลือกทั้งหมด ปักหมุดไว้ถ้าต้องการเข้าถึงเร็วขึ้น
  4. ใช้การปรับปรุงแชท เมนูข้อความ การ์ดอวาตาร์ ปุ่มกล่องข้อความ อีโมจิที่ใช้บ่อย และคำสั่ง Tab จะปรากฏและใช้งานได้ภายในอินเทอร์เฟซแชทของ YouTube

ความเป็นส่วนตัว

เครื่องมือแชทที่เน้นเครื่องของคุณก่อน

ส่วนขยายถูกออกแบบให้การปรับปรุงแชททั่วไปอยู่บนอุปกรณ์ของคุณ การแปลเป็นฟีเจอร์เดียวที่ต้องใช้บริการภายนอก และจะทำงานหลังจากคุณเลือกภาษาปลายทางเท่านั้น

อ่านนโยบายความเป็นส่วนตัวฉบับเต็ม
ขอบเขต
ทำงานเฉพาะบนหน้า YouTube live chat และ live chat replay
การจัดเก็บ
การตั้งค่า ข้อความในกล่อง คำสำคัญที่ติดตาม และจำนวนอีโมจิที่ใช้บ่อยจะอยู่ในพื้นที่เก็บข้อมูลของส่วนขยายเบราว์เซอร์
การแปล
ปิดเป็นค่าเริ่มต้น หากเปิดใช้งาน ข้อความแชตที่เข้ามาหรือข้อความร่างที่คุณเลือกแปลจะถูกส่งไปยัง Google Translate โดยไม่มีคุกกี้หรือข้อมูลเข้าสู่ระบบ YouTube ของคุณ
การติดตาม
ไม่มี analytics ไม่มีการเก็บประวัติการท่องเว็บ ไม่มีการขายข้อมูล และไม่มีเซิร์ฟเวอร์ของส่วนขยายเอง

อ้างอิง

คำสั่งแชทขยายด้วย Tab

พิมพ์คำสั่งในช่องแชทของ YouTube แล้วกด Tab คำสั่งจะไม่ส่งอัตโนมัติ และ slash command ที่ไม่รู้จักจะถูกปล่อยไว้ ใช้ // เพื่อส่ง slash command แบบตัวอักษรจริง

คำสั่งแบบ inline: /mention, /m, /reply, /r, /time, /t, /when, /wh, /timeuntil, /tu, /timesince, /ts, /translate และ /tr สามารถขยายภายในประโยคได้

คำสั่งที่ต้องเป็นข้อความทั้งหมด: /quote, /q, /again, /repeat, /watch, /w, /unwatch, /uw, /focus, /f, /who, /help และคำสั่ง /set... ทั้งหมดควรเป็นข้อความเดียวในช่องพิมพ์

คำสั่งข้อความ

/mention ชื่อย่อ: /m /reply /r
กล่าวถึงผู้เขียนข้อความล่าสุดในกล่องข้อความของคุณ
/quote ชื่อย่อ: /q
ถ้าไม่มี handle จะอ้างอิงข้อความล่าสุดใน inbox ของคุณ ถ้าใช้ @handle จะอ้างอิงข้อความล่าสุดที่เห็นจากผู้ใช้นั้น
/again ชื่อย่อ: /repeat
กู้คืนข้อความล่าสุดที่คุณส่ง
/focus @handle ชื่อย่อ: /f
ถ้าใช้ @handle จะเปิด Focus mode สำหรับผู้ใช้นั้น ถ้าไม่มี handle จะโฟกัสผู้ใช้ล่าสุดที่กล่าวถึงคุณ
/who @handle
เปิดการ์ดผู้ใช้ล่าสุด
/help
แสดงรายการนี้

กล่องข้อความ

/watch keyword ชื่อย่อ: /w
เพิ่มคีย์เวิร์ดหรือวลีที่ต้องการเฝ้าดู ถ้าไม่มีข้อความ จะแสดงรายการที่เฝ้าดูอยู่
/unwatch keyword ชื่อย่อ: /uw
หยุดติดตามคีย์เวิร์ดหรือวลี

เวลาและการแปล

/time ชื่อย่อ: /t ตัวอย่าง: /time utc
แทรกเวลาท้องถิ่นปัจจุบัน หรือเวลาปัจจุบันของสถานที่/เขตเวลาที่รองรับ
alias ที่รองรับ: utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/when 2026-05-29 8pm pt ชื่อย่อ: /wh /timeuntil /tu /timesince /ts
แทรกระยะเวลาจนถึงหรือจากเวลา/วันที่ โดยเลือกสถานที่/เขตเวลาที่รองรับได้
รูปแบบที่รองรับ: 2026-05-29 8pm pt, 2026-06-12 19:00 london, 8pm 2026-06-12 london, tokyo 7pm, and 7pm tokyo.
สถานที่/เขตเวลาที่รองรับ: utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/translate es hello ชื่อย่อ: /tr
แปลข้อความที่กำหนดเองในช่องพิมพ์แชตก่อนส่ง

คำสั่งการตั้งค่า

/settranslateto english ชื่อย่อ: /lang en /lang off
ตั้งค่าภาษาแปล ใช้ off เพื่อปิดการแปล ใช้ /lang โดยไม่ใส่ภาษาเพื่อแสดงเป้าหมายปัจจุบัน
/settranslationdisplay replace|below
ตั้งค่าวิธีแสดงคำแปล
/setsound on|off
ตั้งค่าเสียงกล่องข้อความ
