Primi passi

Installa, aggiorna YouTube una volta se era già aperto, poi usa l'estensione dove la chat live si trova già.

  1. Installa dallo store del browser Usa il pulsante Chrome o Firefox qui sopra. Gli altri browser basati su Chromium possono generalmente usare la build Chrome.
  2. Apri o aggiorna la chat live di YouTube Aggiorna le schede già aperte così l’estensione può collegarsi al frame della live chat. Capirai che funziona quando l’icona della posta in arrivo appare nella chat di YouTube.
  3. Regola le impostazioni complete quando serve L’estensione funziona automaticamente nella live chat. Per le impostazioni complete, clicca l’icona delle estensioni nella barra del browser , e scegli Chat Enhancer per vedere tutte le opzioni. Fissala se vuoi un accesso più rapido.
  4. Usa i miglioramenti della chat Menu messaggi, schede avatar, pulsante posta in arrivo, emoji frequenti e comandi Tab appaiono e si possono usare nell’interfaccia chat di YouTube.

Privacy

Strumenti chat locali per prima cosa

L'estensione è progettata per mantenere sul tuo dispositivo i miglioramenti ordinari della chat. La traduzione è l'unica funzione che richiede un servizio esterno e parte solo dopo aver scelto una lingua di destinazione.

Leggi l'informativa privacy completa
Ambito
Funziona solo sulle pagine di chat live YouTube e replay della chat live.
Archiviazione
Impostazioni, messaggi della posta in arrivo, parole chiave osservate e conteggi degli emoji frequenti restano nello storage dell’estensione del browser.
Traduzione
Disattivata per impostazione predefinita. Se attiva, il testo dei messaggi viene inviato a Google Translate senza cookie o credenziali YouTube.
Tracciamento
Nessuna analisi, raccolta cronologia di navigazione, rivendita dati o server di proprietà dell'estensione.

Riferimento

I comandi chat si espandono con Tab

Digita un comando nel campo chat di YouTube, poi premi Tab. I comandi non vengono inviati automaticamente e i comandi slash sconosciuti restano invariati. Usa // per inviare un comando slash letterale.

Comandi inline: /mention, /m, /reply, /r, /time, /t, /when, /wh, /timeuntil, /tu, /timesince, /ts, /translate e /tr possono espandersi dentro una frase.

Comandi a input intero: /quote, /q, /again, /repeat, /watch, /w, /unwatch, /uw, /focus, /f, /who, /help e tutti i comandi /set... dovrebbero essere l’unico testo nel campo.

Comandi di testo

/mention Alias: /m /reply /r
Menziona l'autore del messaggio più recente nella tua inbox.
/quote Alias: /q
Senza handle, cita il messaggio più recente dell’inbox. Con @handle, cita l’ultimo messaggio visto di quell’utente.
/again Alias: /repeat
Ripristina il tuo ultimo messaggio inviato.
/focus @handle Alias: /f
Con @handle, apre la modalità focus per quell’utente. Senza handle, si concentra sull’ultimo utente che ti ha menzionato.
/who @handle
Apri una scheda utente recente.
/help
Mostra questo elenco.

Posta in arrivo

/watch keyword Alias: /w
Aggiunge parole chiave o frasi osservate. Senza testo, mostra la lista osservata attuale.
/unwatch keyword Alias: /uw
Smetti di monitorare parole chiave o frasi.

Ora e traduzione

/time Alias: /t Esempio: /time utc
Inserisce l’ora locale attuale oppure l’ora attuale per un luogo/fuso orario supportato.
Alias supportati: utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/when 7:45pm Alias: /wh /timeuntil /tu /timesince /ts
Inserisci la durata fino a o da un orario/data.
Formati accettati: 7pm, 19:00, 2026-6-12, 2026-06-12 19:00, tomorrow 8pm, and yesterday 14:30.
/translate es hello Alias: /tr
Traduci testo personalizzato in una lingua.

Comandi impostazioni

/settranslateto english Alias: /lang en /lang off
Imposta la lingua di traduzione. Usa off per disattivare la traduzione. Usa /lang senza lingua per mostrare la destinazione attuale.
/settranslationdisplay replace|below
Imposta come vengono mostrate le traduzioni.
/setsound on|off
Imposta il suono dell'inbox.
