Instal Chat Enhancer

Dapatkan dari Chrome Web Store atau Firefox Add-ons, atau bangun ekstensi dari source untuk pengembangan lokal. Versi Chrome juga dapat bekerja di browser berbasis Chromium lainnya.

Mulai menggunakan

Instal, refresh YouTube sekali jika sudah terbuka, lalu gunakan ekstensi di tempat live chat sudah ada.

  1. Instal dari toko browser Anda Gunakan tombol Chrome atau Firefox di atas. Browser berbasis Chromium lain umumnya dapat memakai build Chrome.
  2. Buka atau refresh live chat YouTube Refresh tab live chat YouTube yang sudah terbuka. Efek startup singkat mungkin muncul; kamu tahu peningkatan chat sudah siap saat ikon kotak masuk muncul di chat YouTube.
  3. Sesuaikan pengaturan lengkap saat diperlukan Ekstensi bekerja otomatis di live chat. Untuk pengaturan lengkap, klik ikon ekstensi di toolbar browser , lalu pilih Chat Enhancer untuk melihat semua opsi. Sematkan jika ingin akses lebih cepat.
  4. Gunakan peningkatan chat Menu pesan, kartu avatar, tombol kotak masuk, emoji sering dipakai, dan perintah Tab muncul dan dapat digunakan di dalam antarmuka chat YouTube.

Privasi

Alat chat yang mengutamakan lokal

Ekstensi dirancang untuk menjaga peningkatan chat rutin tetap di perangkat Anda. Terjemahan adalah satu-satunya fitur yang membutuhkan layanan eksternal, dan hanya berjalan setelah Anda memilih bahasa target.

Baca kebijakan privasi lengkap
Cakupan
Hanya berjalan di halaman live chat YouTube dan replay live chat.
Penyimpanan
Pengaturan, pesan kotak masuk, kata kunci yang dipantau, dan jumlah emoji sering dipakai tetap berada di penyimpanan ekstensi browser.
Terjemahan
Mati secara default. Jika diaktifkan, teks chat masuk atau teks draf yang Anda pilih untuk diterjemahkan dikirim ke Google Translate tanpa cookie atau kredensial YouTube Anda.
Pelacakan
Tidak ada analitik, pengumpulan riwayat browsing, penjualan data, atau server milik ekstensi.

Referensi

Perintah chat diperluas dengan Tab

Ketik perintah di input chat YouTube, lalu tekan Tab. Perintah tidak dikirim otomatis, dan perintah slash yang tidak dikenal dibiarkan. Gunakan // untuk mengirim perintah slash literal.

Perintah inline: /mention, /m, /reply, /r, /time, /t, /when, /wh, /timeuntil, /tu, /timesince, /ts, /translate, dan /tr bisa diperluas di dalam kalimat.

Perintah input penuh: /quote, /q, /again, /repeat, /watch, /w, /unwatch, /uw, /focus, /f, /who, /help, dan semua perintah /set... harus menjadi satu-satunya teks di input.

Perintah teks

/mention Alias: /m /reply /r
Sebut penulis pesan inbox terbaru Anda.
/quote Alias: /q
Tanpa handle, mengutip pesan inbox terbaru. Dengan @handle, mengutip pesan terbaru yang terlihat dari pengguna itu.
/again Alias: /repeat
Pulihkan pesan terakhir yang Anda kirim.
/focus @handle Alias: /f
Dengan @handle, membuka mode focus untuk pengguna itu. Tanpa handle, fokus ke pengguna terakhir yang menyebut Anda.
/who @handle
Buka kartu pengguna terbaru.
/help
Tampilkan daftar ini.

Kotak masuk

/watch keyword Alias: /w
Menambahkan keyword atau frasa yang dipantau. Tanpa teks, menampilkan daftar pantauan saat ini.
/unwatch keyword Alias: /uw
Berhenti memantau kata kunci atau frasa.

Waktu dan terjemahan

/time Alias: /t Contoh: /time utc
Menyisipkan waktu lokal saat ini, atau waktu saat ini untuk tempat/zona waktu yang didukung.
Alias yang didukung: utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/when 2026-05-29 8pm pt Alias: /wh /timeuntil /tu /timesince /ts
Memasukkan durasi hingga atau sejak waktu/tanggal, opsional untuk tempat/zona waktu yang didukung.
Format yang diterima: 2026-05-29 8pm pt, 2026-06-12 19:00 london, 8pm 2026-06-12 london, tokyo 7pm, and 7pm tokyo.
Tempat/zona waktu yang didukung: utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/translate es hello Alias: /tr
Terjemahkan teks khusus di kolom chat sebelum dikirim.

Perintah pengaturan

/settranslateto english Alias: /lang en /lang off
Mengatur bahasa terjemahan. Gunakan off untuk menonaktifkan terjemahan. Gunakan /lang tanpa bahasa untuk menampilkan target saat ini.
/settranslationdisplay replace|below
Atur cara terjemahan ditampilkan.
/setsound on|off
Atur suara inbox.
