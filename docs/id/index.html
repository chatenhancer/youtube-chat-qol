Instal Chat Enhancer

Dapatkan dari Chrome Web Store atau Firefox Add-ons, atau bangun ekstensi dari source untuk pengembangan lokal. Versi Chrome juga dapat bekerja di browser berbasis Chromium lainnya.

Mulai menggunakan

Instal, refresh YouTube sekali jika sudah terbuka, lalu gunakan ekstensi di tempat live chat sudah ada.

  1. Instal dari toko browser Anda Gunakan tombol Chrome atau Firefox di atas. Browser berbasis Chromium lain umumnya dapat memakai build Chrome.
  2. Buka atau refresh live chat YouTube Segarkan tab yang sudah terbuka agar ekstensi dapat terhubung ke frame live chat. Kamu tahu ekstensi sudah bekerja saat ikon inbox muncul di chat YouTube.
  3. Sesuaikan pengaturan lengkap saat diperlukan Ekstensi bekerja otomatis di live chat. Untuk pengaturan lengkap, klik ikon ekstensi di toolbar browser , lalu pilih Chat Enhancer untuk melihat semua opsi. Sematkan jika ingin akses lebih cepat.
  4. Gunakan peningkatan chat Menu pesan, kartu avatar, tombol inbox, emoji sering dipakai, dan perintah Tab muncul dan dapat digunakan di dalam antarmuka chat YouTube.

Privasi

Alat chat yang mengutamakan lokal

Ekstensi dirancang untuk menjaga peningkatan chat rutin tetap di perangkat Anda. Terjemahan adalah satu-satunya fitur yang membutuhkan layanan eksternal, dan hanya berjalan setelah Anda memilih bahasa target.

Baca kebijakan privasi lengkap
Cakupan
Hanya berjalan di halaman live chat YouTube dan replay live chat.
Penyimpanan
Pengaturan, pesan inbox, kata kunci yang dipantau, dan jumlah emoji sering dipakai tetap berada di penyimpanan ekstensi browser.
Terjemahan
Mati secara default. Jika diaktifkan, teks pesan dikirim ke Google Translate tanpa cookie atau kredensial YouTube Anda.
Pelacakan
Tidak ada analitik, pengumpulan riwayat browsing, penjualan data, atau server milik ekstensi.

Referensi

Perintah chat diperluas dengan Tab

Ketik perintah di input chat YouTube, lalu tekan Tab. Perintah tidak dikirim otomatis, dan perintah slash yang tidak dikenal dibiarkan. Gunakan // untuk mengirim perintah slash literal.

Perintah inline: /mention, /reply, /time, dan /timeuntil dapat diperluas di dalam kalimat.

Perintah input penuh: /quote, /again, /repeat, /help, dan semua perintah /set... harus menjadi satu-satunya teks di input.

Perintah teks

/mention or /reply
Masukkan sebutan untuk penulis pesan inbox terbaru Anda.
/quote
Kutip pesan terbaru yang disimpan di inbox Anda.
/again or /repeat
Pulihkan pesan terakhir yang Anda kirim, termasuk konten emoji kaya jika tersedia.
/time utc
Masukkan waktu saat ini untuk zona waktu atau kota yang didukung.
Alias yang didukung: utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/timeuntil 7:45pm
Masukkan waktu tersisa hingga waktu lokal cocok berikutnya.
Format yang diterima: 7, 7am, 7 am, 7:45am, 7:45 am, 19, 19:00, 19:00:30, and 7:45:30 pm.
/help
Buka kartu bantuan perintah di dalam chat YouTube.

Perintah pengaturan

/settranslateto english or /settranslateto off
Atur bahasa target terjemahan. Gunakan off untuk menonaktifkan terjemahan.
/settranslationdisplay replace or /settranslationdisplay below
Ganti pesan asli dengan terjemahan. Gunakan below untuk menampilkan terjemahan di bawah.
/setquotelength 120
Atur panjang kutipan ke 80, 120, 180, atau 240.
/setsound on or /setsound off
Nyalakan atau matikan suara inbox.
/setopenchannelsinpopup on or /setopenchannelsinpopup off
Buka channel di jendela popup. Gunakan off untuk membukanya secara normal.
