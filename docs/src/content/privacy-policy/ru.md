# Политика конфиденциальности

Последнее обновление: 17 июня 2026 г.

Этот перевод предоставлен для удобства. Английская версия на /privacy/ является основным источником.

Большинство функций Chat Enhancer for YouTube работают локально в браузере. Перевод по умолчанию выключен; при включении переводимый текст отправляется в Google Translate. Игры Playground по умолчанию выключены. Если вы включаете и используете их, игровое присутствие, приглашения и игровые действия отправляются в backend Chat Enhancer Playground под сгенерированным именем игрока.

## Данные, хранящиеся в браузере

Расширение использует хранилище браузера для настроек, данных Inbox, отслеживаемых ключевых слов, часто используемых эмодзи, сохранённых пользователей, неотправленных черновиков чата по стримам и статуса вкладок live chat. Псевдонимная идентичность Playground хранится локально, если используется Playground. Последние сообщения профиля и результаты перевода хранятся только в памяти текущей страницы live chat и удаляются при уходе со страницы.

## Данные, отправляемые за пределы браузера

Данные могут отправляться за пределы браузера только при включённом переводе или Playground. Запросы перевода отправляются в Google Translate и не включают cookies или учётные данные YouTube. Playground может получать ключ стрима/видео, сгенерированные публичный ключ и подпись, сгенерированное имя игрока, доступные игры, приглашения и игровые действия, например шахматные ходы. HELP-A-FRIEND! Trivia может отправлять выбранные фрагменты транскрипта YouTube replay и идентификаторы игры в backend Playground, который использует OpenAI для генерации вопросов. Replay Trivia может требовать проверку Cloudflare Turnstile; Cloudflare может получать обычные проверочные данные, например IP-адрес, user agent и результат challenge.

## Управление и ограничения

Вы можете очистить данные расширения в popup с помощью кнопки сброса или удалить расширение из браузера. Расширение не запускает аналитику, не собирает историю браузера и не продаёт пользовательские данные. Кроме описанных выше опциональных игр Playground, оно не отправляет данные на сервер, принадлежащий расширению. Chat Enhancer for YouTube не связан с YouTube или Google.

По вопросам конфиденциальности используйте ссылку e-mail на https://www.chatenhancer.com.
