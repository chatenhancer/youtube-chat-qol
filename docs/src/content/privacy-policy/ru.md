# Конфиденциальность

Последнее обновление: 21 июня 2026 г.

Chat Enhancer for YouTube — это расширение браузера для живого чата YouTube. Оно предназначено для добавления небольших функций чата без замены чата YouTube и без сбора аналитики.

Кратко:

- Большинство функций расширения выполняется локально в вашем браузере.
- Перевод по умолчанию отключен.
- Когда перевод включен, переводимый текст отправляется в Google Translate.
- Игры Playground по умолчанию отключены. Если вы включаете и используете Playground, игровое присутствие, приглашения и игровые действия отправляются на игровой сервер Chat Enhancer Playground под сгенерированным именем игрока.
- Расширение не запускает аналитику, не продает данные и не собирает историю просмотров.

## Где работает расширение

Расширение работает только на страницах живого чата YouTube и повтора живого чата, к которым расширению разрешен доступ.

Расширение использует разрешение для сохранения собственных настроек и данных в вашем браузере. Оно также использует доступ к конкретным сайтам, необходимым для работы его функций: страницам живого чата YouTube, сервису перевода Google Translate и опциональному игровому серверу Chat Enhancer Playground.

Расширение не запрашивает общие разрешения на историю просмотров, чтение вкладок, scripting или web navigation.

## Данные, хранящиеся в вашем браузере

Расширение хранит некоторые данные, чтобы его функции работали между перезагрузками страницы.

Данные, перечисленные в этом разделе, сохраняются расширением в вашем собственном профиле браузера. Они не отправляются в Chat Enhancer, если также не указаны ниже в разделе «Данные, отправляемые за пределы вашего браузера».

- **Настройки:** сохраняются с помощью синхронизированного хранилища расширения браузера (`chrome.storage.sync`). В зависимости от настроек вашего браузера он может синхронизировать эти настройки расширения между вашими собственными установленными браузерами, где выполнен вход.

- **Данные Inbox:** сохраняются с помощью локального хранилища расширения (`chrome.storage.local`). Сюда входят отслеживаемые ключевые слова и до 100 записей inbox на каждый стрим или повтор. Записи Inbox могут включать текст сообщения, имя автора, временную метку, базовые сведения о сообщении YouTube, необходимые для показа источника сохраненного сообщения, сведения о совпадении и информацию об emoji или изображениях, необходимую для корректного показа сохраненного сообщения.

- **Данные частых emoji:** сохраняются с помощью локального хранилища расширения (`chrome.storage.local`). Сюда входят локальные счетчики использования и информация отображения emoji, используемая для создания строки частых emoji.

- **Данные пользователей в закладках:** сохраняются с помощью локального хранилища расширения (`chrome.storage.local`). Сюда входят handle пользователя в закладке, ID канала при наличии и время создания закладки. Пользователи в закладках являются глобальными между стримами в текущем профиле браузера и используются для отображения цветных колец аватаров.

- **Неотправленные черновики чата:** сохраняются с помощью локального хранилища расширения (`chrome.storage.local`) для каждого стрима. Они восстанавливаются после обновления страницы. Черновики удаляются, когда поле ввода чата очищается, сообщение отправляется или данные расширения сбрасываются.

- **Данные идентичности Playground:** сохраняются с помощью локального хранилища расширения (`chrome.storage.local`), если используется Playground. Это случайно сгенерированная локальная идентичность Playground, используемая для распознавания той же установки браузера при повторном подключении к Playground. Это не ваша идентичность YouTube.

- **Недавние сообщения профиля, состояние команд и результаты перевода:** хранятся только в памяти текущей страницы живого чата. Они очищаются, когда вы покидаете или обновляете страницу чата.

## Данные, отправляемые за пределы вашего браузера

Перевод чата, перевод черновиков и игры Playground по умолчанию отключены.

Когда функции перевода или Playground включены и используются, данные могут отправляться этим сервисам:

- **Google Translate по адресу `https://translate.googleapis.com/translate_a/single`**

  Перевод чата отправляет текст сообщений, который виден в живом чате и подходит для перевода, пока перевод включен. Перевод черновиков отправляет текст черновика, который вы выбираете для перевода из поля чата.

  Запросы перевода включают текст для перевода и целевой язык. Расширение не отправляет ваши cookies YouTube или учетные данные YouTube вместе с запросами перевода.

  Доступ к Google Translate через `translate.googleapis.com` неофициальный и может быть ограничен по частоте, изменен или недоступен.

- <span id="playground"></span>**Chat Enhancer Playground по адресу `https://playground.chatenhancer.com`**

  Playground по умолчанию отключен. Если вы включаете Playground и используете панель игр, расширение подключается к игровому серверу Chat Enhancer Playground, чтобы пользователи, выбравшие участие в том же стриме, могли видеть доступность, обмениваться приглашениями и играть.

  Сообщения Playground могут включать идентификатор стрима или видео YouTube, вашу сгенерированную идентичность игрока Playground, ваше сгенерированное имя игрока, ваш список доступных игр, приглашения и ответы на приглашения, а также игровые действия, например шахматные ходы.

  Playground не отправляет текст живого чата, ваше отображаемое имя YouTube, URL вашего аватара YouTube, cookies YouTube или учетные данные YouTube на игровой сервер Playground.

  Отдельно генерация вопросов HELP-A-FRIEND! Trivia может отправлять выбранные фрагменты публичных транскриптов видео YouTube и идентификаторы игры на игровой сервер Playground. Эти фрагменты берутся из транскрипта видео, а не из живого чата. Сервер использует OpenAI для генерации trivia-вопросов из этих фрагментов.

  Генерация Replay Trivia может требовать проверки Cloudflare Turnstile на `https://playground.chatenhancer.com`. Cloudflare может получать обычные проверочные данные, такие как IP-адрес, информация о браузере и устройстве, а также результат challenge.

  Как любой веб-сервис, игровой сервер Playground может получать обычную информацию о подключении, такую как IP-адрес и информация о браузере/устройстве, от браузера или сетевого провайдера.

## Управление данными

Вы можете очистить данные расширения во всплывающем окне расширения с помощью кнопки сброса. Это очищает локальные данные расширения и синхронизированные настройки расширения, а затем восстанавливает настройки по умолчанию.

Вы также можете удалить расширение из браузера. В зависимости от браузера удаление расширения также может удалить его локальное хранилище расширения.

## Чего Chat Enhancer не делает

Расширение не запускает аналитику.

Расширение не собирает историю просмотров.

Расширение не продает пользовательские данные.

За исключением описанных выше opt-in функций Playground, расширение не отправляет данные на сервер Chat Enhancer.

Расширение не хранит недавние сообщения профиля или результаты перевода после того, как вы покидаете или обновляете страницу живого чата.

Chat Enhancer for YouTube не связан с YouTube или Google.

По вопросам конфиденциальности используйте ссылку электронной почты на https://www.chatenhancer.com.
