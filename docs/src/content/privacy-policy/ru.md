# Конфиденциальность

Последнее обновление: 17 июня 2026 г.

Chat Enhancer for YouTube — это расширение браузера для живого чата YouTube. Оно предназначено для добавления небольших функций чата без замены чата YouTube и без сбора аналитики.

Кратко:

- Большинство функций расширения выполняется локально в вашем браузере.
- Перевод по умолчанию отключен.
- Когда перевод включен, переводимый текст отправляется в Google Translate.
- Игры Playground по умолчанию отключены. Если вы включаете и используете Playground, игровое присутствие, приглашения и игровые действия отправляются на backend Chat Enhancer Playground под сгенерированным именем игрока.
- Расширение не запускает аналитику, не продает данные и не собирает историю просмотров.

## Где работает расширение

Расширение работает только на страницах живого чата YouTube и повтора живого чата, соответствующих manifest расширения.

Расширение использует разрешение браузера `storage`, а также host access для страниц живого чата YouTube, переводческого endpoint Google и опционального backend Playground. Оно не запрашивает общие разрешения на историю просмотров, чтение вкладок, scripting или web navigation.

## Данные, хранящиеся в вашем браузере

Расширение хранит некоторые данные, чтобы его функции работали между перезагрузками страницы.

- **Настройки хранятся с помощью `chrome.storage.sync`:** в зависимости от настроек вашего браузера он может синхронизировать эти настройки расширения между вашими собственными установленными браузерами, где выполнен вход.

- **Данные Inbox хранятся с помощью `chrome.storage.local`:** сюда входят отслеживаемые ключевые слова и до 100 записей inbox на каждый стрим или повтор. Записи Inbox могут включать текст сообщения, имя автора, временную метку, метаданные сообщения/источника YouTube, метаданные совпадения и данные отображения emoji/изображений, необходимые для показа сохраненного сообщения.

- **Данные частых emoji хранятся с помощью `chrome.storage.local`:** сюда входят локальные счетчики использования и метаданные отображения emoji, используемые для создания строки частых emoji.

- **Данные пользователей в закладках хранятся с помощью `chrome.storage.local`:** сюда входят handle пользователя в закладке, ID канала при наличии и время создания закладки. Пользователи в закладках являются глобальными между стримами в текущем профиле браузера и используются для отображения цветных колец аватаров.

- **Неотправленные черновики чата хранятся с помощью `chrome.storage.local` для каждого стрима:** они восстанавливаются после обновления страницы. Черновики удаляются, когда поле ввода чата очищается, сообщение отправляется или данные расширения сбрасываются.

- **Статус вкладки живого чата хранится с помощью `chrome.storage.local`:** он ограничен ID вкладок браузера и временными метками последней активности для недавно активных вкладок живого чата YouTube и используется, чтобы показать, подключено расширение сейчас или отключено. Эти записи истекают через 12 часов.

- **Данные идентичности Playground хранятся с помощью `chrome.storage.local`, если используется Playground:** это сгенерированная пара публичного/приватного ключей, используемая для подписи challenge подключения Playground, чтобы одна и та же установка браузера могла сохранять одну и ту же псевдонимную идентичность Playground. Это не ваша идентичность YouTube.

- **Недавние сообщения профиля, состояние команд и результаты перевода хранятся только в памяти текущей страницы живого чата. Они очищаются при выгрузке страницы.**

## Данные, отправляемые за пределы вашего браузера

Перевод чата и перевод черновиков по умолчанию отключены.

Когда включены функции перевода или Playground, данные могут отправляться этим сервисам:

- **Google Translate по адресу `https://translate.googleapis.com/translate_a/single`**

  Перевод чата отправляет подходящий видимый и входящий текст сообщений чата. Перевод черновиков отправляет текст черновика, который вы выбираете для перевода из поля чата.

  Запросы перевода включают текст для перевода и целевой язык. Расширение не отправляет ваши cookies YouTube или учетные данные YouTube вместе с запросами перевода.

  Доступ к Google Translate через `translate.googleapis.com` неофициальный и может быть ограничен по частоте, изменен или недоступен.

- **Chat Enhancer Playground по адресу `https://playground.chatenhancer.com`**

  Playground по умолчанию отключен. Если вы включаете Playground и используете панель игр, расширение подключается к backend Playground, чтобы пользователи, выбравшие участие в том же стриме, могли видеть доступность, обмениваться приглашениями и играть.

  Сообщения Playground могут включать ключ стрима/видео, ваш сгенерированный публичный ключ Playground и подпись, ваше сгенерированное имя игрока, ваш список доступных игр, приглашения и ответы на приглашения, а также игровые действия, например шахматные ходы.

  Генерация вопросов HELP-A-FRIEND! Trivia может отправлять выбранные фрагменты transcript повтора YouTube и идентификаторы игры на backend Playground. Backend использует OpenAI для генерации trivia-вопросов из этих фрагментов.

  Генерация Replay Trivia может требовать проверки Cloudflare Turnstile на `https://playground.chatenhancer.com`. Cloudflare может получать обычные проверочные данные, такие как IP-адрес, user agent и результат challenge.

  Playground не отправляет на backend Playground текст живого чата, ваше отображаемое имя YouTube, URL вашего аватара YouTube, cookies YouTube или учетные данные YouTube.

  Как любой веб-сервис, backend Playground может получать обычные метаданные подключения, такие как IP-адрес и user agent, от браузера или сетевого провайдера.

## Управление данными

Вы можете очистить данные расширения во всплывающем окне расширения с помощью кнопки сброса. Это очищает локальные данные расширения и синхронизированные настройки расширения, а затем восстанавливает настройки по умолчанию.

Вы также можете удалить расширение из браузера. В зависимости от браузера удаление расширения также может удалить его локальное хранилище расширения.

## Что не собирается

Расширение не запускает аналитику.

Расширение не собирает историю просмотров.

Расширение не продает пользовательские данные.

За исключением описанных выше opt-in игр Playground, расширение не отправляет данные на сервер, принадлежащий расширению.

Расширение не хранит недавние сообщения профиля или результаты перевода после выгрузки страницы живого чата.

Chat Enhancer for YouTube не связан с YouTube или Google.

По вопросам конфиденциальности используйте ссылку электронной почты на https://www.chatenhancer.com.
