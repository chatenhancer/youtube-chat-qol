# プライバシーポリシー

最終更新日: 2026年6月17日

この翻訳は便宜のために提供されています。/privacy/ の英語版が主な参照です。

Chat Enhancer for YouTube のほとんどの機能はブラウザ内でローカルに実行されます。翻訳は既定でオフです。有効にすると、翻訳対象のテキストが Google Translate に送信されます。Playground ゲームも既定でオフです。有効にして使用すると、ゲームの在席情報、招待、ゲーム操作が、生成されたプレイヤー名の下で Chat Enhancer Playground バックエンドに送信されます。

## ブラウザに保存されるデータ

拡張機能は、設定、Inbox データ、監視キーワード、よく使う絵文字、ブックマークしたユーザー、ストリームごとの未送信チャット下書き、ライブチャットタブの状態を保存するためにブラウザストレージを使用します。Playground を使用する場合、仮名の Playground ID がローカルに保存されます。最近のプロフィールメッセージと翻訳結果は、現在のライブチャットページのメモリにのみ保持され、ページを離れると消去されます。

## ブラウザ外へ送信されるデータ

データがブラウザ外へ送信される可能性があるのは、翻訳または Playground を有効にした場合のみです。翻訳リクエストは Google Translate に送信され、YouTube の Cookie や認証情報は含まれません。Playground は、ストリーム/動画キー、生成された公開鍵と署名、生成されたプレイヤー名、利用可能なゲーム、招待、チェスの手などのゲーム操作を受け取る場合があります。HELP-A-FRIEND! Trivia は、選択された YouTube リプレイの文字起こし抜粋とゲーム識別子を Playground バックエンドへ送信し、バックエンドは OpenAI を使用して質問を生成する場合があります。Replay Trivia では Cloudflare Turnstile の確認が必要になることがあり、Cloudflare は IP アドレス、ユーザーエージェント、チャレンジ結果など通常の確認データを受け取る場合があります。

## 管理と制限

拡張機能のポップアップにあるリセットボタンで拡張機能データを消去できます。また、ブラウザから拡張機能を削除することもできます。拡張機能は分析を実行せず、閲覧履歴を収集せず、ユーザーデータを販売しません。上記のオプトイン Playground ゲームを除き、拡張機能所有のサーバーへデータを送信しません。Chat Enhancer for YouTube は YouTube または Google と提携していません。

プライバシーに関する質問は https://www.chatenhancer.com のメールリンクを使用してください。
