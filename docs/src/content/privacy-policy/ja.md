# プライバシー

最終更新日: 2026年6月21日

Chat Enhancer for YouTube は、YouTube ライブチャット向けのブラウザー拡張機能です。YouTube チャットを置き換えたり分析データを収集したりせずに、小さなチャット機能を追加することを目的としています。

要点:

- 拡張機能のほとんどの機能は、あなたのブラウザー内でローカルに実行されます。
- 翻訳はデフォルトでオフです。
- 翻訳を有効にすると、翻訳対象のテキストが Google Translate に送信されます。
- Playground ゲームはデフォルトでオフです。Playground を有効にして使用すると、ゲームのプレゼンス、招待、ゲーム操作が、生成されたプレイヤー名の下で Chat Enhancer Playground ゲームサーバーに送信されます。
- 拡張機能は分析を実行せず、データを販売せず、閲覧履歴を収集しません。

## 拡張機能が実行される場所

拡張機能は、アクセスが許可された YouTube ライブチャットおよびライブチャットリプレイページでのみ実行されます。

拡張機能は、自身の設定とデータをブラウザーに保存するための権限を使用します。また、機能の動作に必要な特定のウェブサイトへのアクセスも使用します。対象は YouTube ライブチャットページ、Google Translate の翻訳サービス、任意参加の Chat Enhancer Playground ゲームサーバーです。

拡張機能は、一般的な閲覧履歴、タブ読み取り、スクリプト実行、Web ナビゲーションの権限は要求しません。

## ブラウザーに保存されるデータ

拡張機能は、ページの再読み込みをまたいで機能が動作するように一部のデータを保存します。

このセクションに記載されたデータは、拡張機能によってあなた自身のブラウザープロファイルに保存されます。下の「ブラウザー外へ送信されるデータ」セクションにも記載されていない限り、Chat Enhancer には送信されません。

- **設定:** ブラウザーの同期された拡張機能ストレージ (`chrome.storage.sync`) に保存されます。ブラウザー設定によっては、ブラウザーがこれらの拡張機能設定を、あなた自身がログインしているブラウザーインストール間で同期する場合があります。

- **Inbox データ:** ローカル拡張機能ストレージ (`chrome.storage.local`) に保存されます。これには監視キーワードと、ストリームまたはリプレイごとに最大 100 件の Inbox レコードが含まれます。Inbox レコードには、メッセージ本文、投稿者名、タイムスタンプ、保存メッセージの出どころを示すために必要な基本的な YouTube メッセージ詳細、一致の詳細、保存メッセージを正しく表示するために必要な絵文字または画像情報が含まれる場合があります。

- **よく使う絵文字データ:** ローカル拡張機能ストレージ (`chrome.storage.local`) に保存されます。これには、よく使う絵文字行を作成するために使われるローカル使用回数と絵文字表示情報が含まれます。

- **ブックマークデータ:** ローカル拡張機能ストレージ (`chrome.storage.local`) に保存されます。保存したメッセージのテキストと絵文字表示情報、投稿者名、アバター URL、利用可能な場合はチャンネル ID、メッセージと保存の日時、配信のタイトルと URL が含まれることがあります。ブックマークは現在のブラウザープロファイル内で配信をまたいで利用できます。

- **アバターリングデータ:** ローカル拡張機能ストレージ (`chrome.storage.local`) に保存されます。最近のメッセージのプロフィールから明示的にリングを追加したユーザーについて、投稿者名、リングを追加した日時、配信 URL、利用可能な場合はアバター URL、チャンネル ID、配信タイトルが含まれます。この選択は現在のブラウザープロファイル内で配信をまたいで利用でき、一致するアバターを装飾するためだけに使用されます。ユーザーがオンラインかどうかは確認しません。

- **未送信のチャット下書き:** ストリームごとにローカル拡張機能ストレージ (`chrome.storage.local`) に保存されます。ページ更新後に復元されます。下書きは、チャット入力が消去されたとき、メッセージが送信されたとき、または拡張機能データがリセットされたときに削除されます。

- **Playground ID データ:** Playground を使用する場合、ローカル拡張機能ストレージ (`chrome.storage.local`) に保存されます。これはランダムに生成されたローカルの Playground ID で、Playground に再接続したときに同じブラウザーインストールを認識するために使われます。これはあなたの YouTube ID ではありません。

- **最近のプロフィールメッセージ、コマンド状態、翻訳結果:** 現在のライブチャットページ用にメモリ内だけに保持されます。チャットページを離れるか更新すると消去されます。

## ブラウザー外へ送信されるデータ

チャット翻訳、下書き翻訳、Playground ゲームはデフォルトでオフです。

翻訳または Playground 機能を有効にして使用すると、データが次のサービスに送信される場合があります。

- **`https://translate.googleapis.com/translate_a/single` の Google Translate**

  チャット翻訳は、翻訳が有効な間にライブチャットで表示され、翻訳対象となるチャットメッセージ本文を送信します。下書き翻訳は、チャットボックスから翻訳することを選んだ下書きテキストを送信します。

  翻訳リクエストには、翻訳するテキストと対象言語が含まれます。拡張機能は、翻訳リクエストに YouTube Cookie や YouTube 認証情報を送信しません。

  `translate.googleapis.com` 経由の Google Translate アクセスは非公式であり、レート制限、変更、または利用不能になる場合があります。

- <span id="playground"></span>**`https://playground.chatenhancer.com` の Chat Enhancer Playground**

  Playground はデフォルトでオフです。Playground を有効にしてゲームパネルを使用すると、拡張機能は Chat Enhancer Playground ゲームサーバーに接続し、同じストリームでオプトインしたユーザーが空き状況を確認し、招待を交換し、ゲームをプレイできるようにします。

  Playground メッセージには、YouTube ストリームまたは動画の識別子、生成された Playground プレイヤー ID、生成されたプレイヤー名、利用可能なゲームリスト、招待と招待応答、チェスの手などのゲーム操作が含まれる場合があります。

  Playground は、ライブチャットメッセージ本文、あなたの YouTube 表示名、YouTube アバター URL、YouTube Cookie、YouTube 認証情報を Playground ゲームサーバーへ送信しません。

  これとは別に、HELP-A-FRIEND! Trivia の質問生成では、選択された公開 YouTube 動画トランスクリプトの抜粋とゲーム識別子が Playground ゲームサーバーに送信される場合があります。これらの抜粋は動画のトランスクリプトから取得されるもので、ライブチャットからではありません。サーバーは OpenAI を使用して、それらの抜粋から trivia 質問を生成します。

  Replay Trivia の生成では、`https://playground.chatenhancer.com` で Cloudflare Turnstile 検証が必要になる場合があります。Cloudflare は IP アドレス、ブラウザーおよびデバイス情報、チャレンジ結果などの通常の検証データを受け取る場合があります。

  他の Web サービスと同様に、Playground ゲームサーバーはブラウザーまたはネットワークプロバイダーから、IP アドレスやブラウザー/デバイス情報などの通常の接続情報を受け取る場合があります。

## データの管理

拡張機能ポップアップのリセットボタンを使って、拡張機能データを消去できます。これにより、ローカルの拡張機能データと同期された拡張機能設定が消去され、デフォルト設定に戻ります。

ブラウザーから拡張機能を削除することもできます。ブラウザーによっては、拡張機能の削除によりローカルの拡張機能ストレージも削除される場合があります。

## Chat Enhancer が行わないこと

拡張機能は分析を実行しません。

拡張機能は閲覧履歴を収集しません。

拡張機能はユーザーデータを販売しません。

上記の任意参加の Playground 機能を除き、拡張機能は Chat Enhancer サーバーにデータを送信しません。

拡張機能は、ライブチャットページを離れるか更新した後に最近のプロフィールメッセージや翻訳結果を保存しません。

Chat Enhancer for YouTube は YouTube または Google と提携していません。

プライバシーに関する質問は、https://www.chatenhancer.com のメールリンクを使用してください。
