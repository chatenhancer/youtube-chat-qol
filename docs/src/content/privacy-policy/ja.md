# プライバシー

最終更新日: 2026年6月17日

Chat Enhancer for YouTube は、YouTube ライブチャット向けのブラウザー拡張機能です。YouTube チャットを置き換えたり分析データを収集したりせずに、小さなチャット機能を追加することを目的としています。

要点:

- 拡張機能のほとんどの機能は、あなたのブラウザー内でローカルに実行されます。
- 翻訳はデフォルトでオフです。
- 翻訳を有効にすると、翻訳対象のテキストが Google Translate に送信されます。
- Playground ゲームはデフォルトでオフです。Playground を有効にして使用すると、ゲームのプレゼンス、招待、ゲーム操作が、生成されたプレイヤー名の下で Chat Enhancer Playground バックエンドに送信されます。
- 拡張機能は分析を実行せず、データを販売せず、閲覧履歴を収集しません。

## 拡張機能が実行される場所

拡張機能は、拡張機能の manifest に一致する YouTube ライブチャットおよびライブチャットリプレイページでのみ実行されます。

拡張機能は、ブラウザーの `storage` 権限に加え、YouTube ライブチャットページ、Google の翻訳エンドポイント、任意参加の Playground バックエンドへのホストアクセスを使用します。一般的な閲覧履歴、タブ読み取り、スクリプト実行、Web ナビゲーションの権限は要求しません。

## ブラウザーに保存されるデータ

拡張機能は、ページの再読み込みをまたいで機能が動作するように一部のデータを保存します。

- **設定は `chrome.storage.sync` に保存されます:** ブラウザー設定によっては、ブラウザーがこれらの拡張機能設定を、あなた自身がログインしているブラウザーインストール間で同期する場合があります。

- **Inbox データは `chrome.storage.local` に保存されます:** これには監視キーワードと、ストリームまたはリプレイごとに最大 100 件の Inbox レコードが含まれます。Inbox レコードには、メッセージ本文、投稿者名、タイムスタンプ、YouTube メッセージ/ソースのメタデータ、一致メタデータ、保存メッセージを表示するために必要な絵文字/画像表示データが含まれる場合があります。

- **よく使う絵文字データは `chrome.storage.local` に保存されます:** これには、よく使う絵文字行を作成するために使われるローカル使用回数と絵文字表示メタデータが含まれます。

- **ブックマークしたユーザーデータは `chrome.storage.local` に保存されます:** これには、ブックマークしたユーザーのハンドル、利用可能な場合のチャンネル ID、ブックマーク作成時刻が含まれます。ブックマークしたユーザーは現在のブラウザープロファイル内のストリーム間で共通で、色付きアバターリングの表示に使われます。

- **未送信のチャット下書きは、ストリームごとに `chrome.storage.local` に保存されます:** ページ更新後に復元されます。下書きは、チャット入力が消去されたとき、メッセージが送信されたとき、または拡張機能データがリセットされたときに削除されます。

- **ライブチャットタブの状態は `chrome.storage.local` に保存されます:** これは最近アクティブだった YouTube ライブチャットタブのブラウザータブ ID と最終確認タイムスタンプに限定され、拡張機能が現在接続中か切断中かを表示するために使われます。これらのレコードは 12 時間後に期限切れになります。

- **Playground を使用する場合、Playground ID データは `chrome.storage.local` に保存されます:** これは Playground 接続チャレンジに署名するために生成される公開/秘密鍵ペアで、同じブラウザーインストールが同じ仮名の Playground ID を維持できるようにするものです。これはあなたの YouTube ID ではありません。

- **最近のプロフィールメッセージ、コマンド状態、翻訳結果は、現在のライブチャットページ用にメモリ内だけに保持されます。ページがアンロードされると消去されます。**

## ブラウザー外へ送信されるデータ

チャット翻訳と下書き翻訳はデフォルトでオフです。

翻訳または Playground 機能を有効にすると、データが次のサービスに送信される場合があります。

- **`https://translate.googleapis.com/translate_a/single` の Google Translate**

  チャット翻訳は、対象となる表示中および受信中のチャットメッセージ本文を送信します。下書き翻訳は、チャットボックスから翻訳することを選んだ下書きテキストを送信します。

  翻訳リクエストには、翻訳するテキストと対象言語が含まれます。拡張機能は、翻訳リクエストに YouTube Cookie や YouTube 認証情報を送信しません。

  `translate.googleapis.com` 経由の Google Translate アクセスは非公式であり、レート制限、変更、または利用不能になる場合があります。

- **`https://playground.chatenhancer.com` の Chat Enhancer Playground**

  Playground はデフォルトでオフです。Playground を有効にしてゲームパネルを使用すると、拡張機能は Playground バックエンドに接続し、同じストリームでオプトインしたユーザーが空き状況を確認し、招待を交換し、ゲームをプレイできるようにします。

  Playground メッセージには、ストリーム/動画キー、生成された Playground 公開鍵と署名、生成されたプレイヤー名、利用可能なゲームリスト、招待と招待応答、チェスの手などのゲーム操作が含まれる場合があります。

  HELP-A-FRIEND! Trivia の質問生成では、選択された YouTube リプレイトランスクリプトの抜粋とゲーム識別子が Playground バックエンドに送信される場合があります。バックエンドは OpenAI を使用して、それらの抜粋から trivia 質問を生成します。

  Replay Trivia の生成では、`https://playground.chatenhancer.com` で Cloudflare Turnstile 検証が必要になる場合があります。Cloudflare は IP アドレス、ユーザーエージェント、チャレンジ結果などの通常の検証データを受け取る場合があります。

  Playground は、ライブチャットメッセージ本文、あなたの YouTube 表示名、YouTube アバター URL、YouTube Cookie、YouTube 認証情報を Playground バックエンドへ送信しません。

  他の Web サービスと同様に、Playground バックエンドはブラウザーまたはネットワークプロバイダーから、IP アドレスやユーザーエージェントなどの通常の接続メタデータを受け取る場合があります。

## データの管理

拡張機能ポップアップのリセットボタンを使って、拡張機能データを消去できます。これにより、ローカルの拡張機能データと同期された拡張機能設定が消去され、デフォルト設定に戻ります。

ブラウザーから拡張機能を削除することもできます。ブラウザーによっては、拡張機能の削除によりローカルの拡張機能ストレージも削除される場合があります。

## 収集されないもの

拡張機能は分析を実行しません。

拡張機能は閲覧履歴を収集しません。

拡張機能はユーザーデータを販売しません。

上記の任意参加の Playground ゲームを除き、拡張機能は拡張機能所有のサーバーにデータを送信しません。

拡張機能は、ライブチャットページのアンロード後に最近のプロフィールメッセージや翻訳結果を保存しません。

Chat Enhancer for YouTube は YouTube または Google と提携していません。

プライバシーに関する質問は、https://www.chatenhancer.com のメールリンクを使用してください。
