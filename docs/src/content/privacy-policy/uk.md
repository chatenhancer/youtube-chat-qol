# Політика приватності

Останнє оновлення: 17 червня 2026 року

Цей переклад надано для зручності. Англійська версія на /privacy/ є основним джерелом.

Більшість функцій Chat Enhancer for YouTube працюють локально в браузері. Переклад вимкнено за замовчуванням; якщо його ввімкнути, текст для перекладу надсилається до Google Translate. Ігри Playground вимкнено за замовчуванням. Якщо ви їх увімкнете й використовуватимете, ігрова присутність, запрошення та дії гри надсилатимуться до backend Chat Enhancer Playground під згенерованим іменем гравця.

## Дані, що зберігаються в браузері

Розширення використовує сховище браузера для налаштувань, даних Inbox, відстежуваних ключових слів, частих емодзі, збережених користувачів, ненадісланих чернеток чату для кожного стріму та статусу вкладок live chat. Псевдонімна ідентичність Playground зберігається локально, якщо використовується Playground. Останні повідомлення профілю й результати перекладу зберігаються лише в пам’яті поточної сторінки live chat і видаляються після виходу зі сторінки.

## Дані, що надсилаються за межі браузера

Дані можуть надсилатися за межі браузера лише коли ввімкнено переклад або Playground. Запити перекладу надходять до Google Translate і не містять cookie або облікових даних YouTube. Playground може отримувати ключ стріму/відео, згенеровані публічний ключ і підпис, згенероване ім’я гравця, доступні ігри, запрошення та дії гри, як-от шахові ходи. HELP-A-FRIEND! Trivia може надсилати вибрані фрагменти транскрипту YouTube replay та ідентифікатори гри до backend Playground, який використовує OpenAI для створення питань. Replay Trivia може потребувати перевірки Cloudflare Turnstile, а Cloudflare може отримувати звичайні дані перевірки, як-от IP-адресу, user agent і результат виклику.

## Контролі й обмеження

Ви можете очистити дані розширення з popup за допомогою кнопки скидання або видалити розширення з браузера. Розширення не запускає аналітику, не збирає історію перегляду й не продає дані користувачів. Окрім описаних вище опційних ігор Playground, воно не надсилає дані на сервер, що належить розширенню. Chat Enhancer for YouTube не пов’язаний із YouTube або Google.

Для запитань щодо приватності використовуйте посилання e-mail на https://www.chatenhancer.com.
