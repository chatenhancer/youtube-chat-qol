# ความเป็นส่วนตัว

อัปเดตล่าสุด: 21 มิถุนายน 2026

Chat Enhancer for YouTube เป็นส่วนขยายเบราว์เซอร์สำหรับแชทสดของ YouTube ออกแบบมาเพื่อเพิ่มฟีเจอร์แชทเล็ก ๆ โดยไม่แทนที่แชทของ YouTube หรือเก็บข้อมูลวิเคราะห์

สรุปสั้น ๆ:

- ฟีเจอร์ส่วนใหญ่ของส่วนขยายทำงานภายในเบราว์เซอร์ของคุณ
- การแปลปิดอยู่ตามค่าเริ่มต้น
- เมื่อเปิดใช้การแปล ข้อความที่ถูกแปลจะถูกส่งไปยัง Google Translate
- เกม Playground ปิดอยู่ตามค่าเริ่มต้น หากคุณเปิดใช้และใช้ Playground สถานะการเล่นเกม คำเชิญ และการกระทำในเกมจะถูกส่งไปยัง backend ของ Chat Enhancer Playground ภายใต้ชื่อผู้เล่นที่สร้างขึ้น
- ส่วนขยายไม่เรียกใช้การวิเคราะห์ ไม่ขายข้อมูล และไม่เก็บประวัติการท่องเว็บ

## ส่วนขยายทำงานที่ไหน

ส่วนขยายทำงานเฉพาะบนหน้าแชทสดของ YouTube และหน้า replay แชทสดที่ตรงกับ manifest ของส่วนขยาย

ส่วนขยายใช้สิทธิ์ `storage` ของเบราว์เซอร์ รวมถึง host access สำหรับหน้าแชทสดของ YouTube, endpoint การแปลของ Google และ backend Playground แบบเลือกเข้าร่วม ส่วนขยายไม่ขอสิทธิ์ทั่วไปสำหรับประวัติการท่องเว็บ การอ่านแท็บ scripting หรือ web navigation

## ข้อมูลที่เก็บในเบราว์เซอร์ของคุณ

ส่วนขยายเก็บข้อมูลบางอย่างเพื่อให้ฟีเจอร์ทำงานได้ระหว่างการโหลดหน้าใหม่

- **การตั้งค่าถูกเก็บด้วย `chrome.storage.sync`:** ขึ้นอยู่กับการตั้งค่าเบราว์เซอร์ของคุณ เบราว์เซอร์อาจซิงค์การตั้งค่าส่วนขยายเหล่านี้ระหว่างการติดตั้งเบราว์เซอร์ของคุณที่ลงชื่อเข้าใช้ไว้

- **ข้อมูล Inbox ถูกเก็บด้วย `chrome.storage.local`:** รวมถึงคีย์เวิร์ดที่เฝ้าดูและรายการ inbox สูงสุด 100 รายการต่อ stream หรือ replay รายการ Inbox อาจมีข้อความ ชื่อผู้เขียน timestamp, metadata ของข้อความ/แหล่งที่มาจาก YouTube, metadata การจับคู่ และข้อมูลแสดงผล emoji/รูปภาพที่จำเป็นสำหรับแสดงข้อความที่บันทึกไว้

- **ข้อมูล emoji ที่ใช้บ่อยถูกเก็บด้วย `chrome.storage.local`:** รวมถึงจำนวนการใช้งานในเครื่องและ metadata การแสดงผล emoji ที่ใช้สร้างแถว emoji ที่ใช้บ่อย

- **ข้อมูลผู้ใช้ที่บุ๊กมาร์กถูกเก็บด้วย `chrome.storage.local`:** รวมถึง handle ของผู้ใช้ที่บุ๊กมาร์ก, channel ID เมื่อมี และเวลาที่สร้างบุ๊กมาร์ก ผู้ใช้ที่บุ๊กมาร์กจะใช้ร่วมกันข้าม stream ในโปรไฟล์เบราว์เซอร์ปัจจุบัน และใช้เพื่อแสดงวงแหวน avatar สีต่าง ๆ

- **ร่างแชทที่ยังไม่ได้ส่งถูกเก็บด้วย `chrome.storage.local` ต่อ stream:** จะถูกกู้คืนหลังรีเฟรชหน้า ร่างจะถูกลบเมื่อช่องแชทถูกล้าง ข้อความถูกส่ง หรือข้อมูลส่วนขยายถูกรีเซ็ต

- **ข้อมูลตัวตน Playground ถูกเก็บด้วย `chrome.storage.local` หากใช้ Playground:** เป็นคู่กุญแจ public/private ที่สร้างขึ้นเพื่อเซ็น challenge การเชื่อมต่อ Playground เพื่อให้การติดตั้งเบราว์เซอร์เดิมคงตัวตน Playground แบบนามแฝงเดิมได้ นี่ไม่ใช่ตัวตน YouTube ของคุณ

- **ข้อความโปรไฟล์ล่าสุด สถานะคำสั่ง และผลการแปลจะถูกเก็บไว้ในหน่วยความจำสำหรับหน้าแชทสดปัจจุบันเท่านั้น และจะถูกล้างเมื่อหน้านั้น unload**

## ข้อมูลที่ส่งออกนอกเบราว์เซอร์ของคุณ

การแปลแชทและการแปลร่างปิดอยู่ตามค่าเริ่มต้น

เมื่อเปิดใช้ฟีเจอร์การแปลหรือ Playground ข้อมูลอาจถูกส่งไปยังบริการเหล่านี้:

- **Google Translate ที่ `https://translate.googleapis.com/translate_a/single`**

  การแปลแชทส่งข้อความแชทที่มองเห็นและข้อความขาเข้าที่เข้าเกณฑ์ การแปลร่างส่งข้อความร่างที่คุณเลือกแปลจากกล่องแชท

  คำขอแปลมีข้อความที่จะถูกแปลและภาษาปลายทาง ส่วนขยายไม่ส่ง cookies ของ YouTube หรือ credentials ของ YouTube ของคุณไปกับคำขอแปล

  การเข้าถึง Google Translate ผ่าน `translate.googleapis.com` เป็นแบบไม่เป็นทางการ และอาจถูกจำกัด เปลี่ยนแปลง หรือไม่พร้อมใช้งาน

- **Chat Enhancer Playground ที่ `https://playground.chatenhancer.com`**

  Playground ปิดอยู่ตามค่าเริ่มต้น หากคุณเปิดใช้ Playground และใช้แผงเกม ส่วนขยายจะเชื่อมต่อกับ backend ของ Playground เพื่อให้ผู้ใช้ที่เลือกเข้าร่วมใน stream เดียวกันเห็นสถานะพร้อมเล่น แลกเปลี่ยนคำเชิญ และเล่นเกมได้

  ข้อความ Playground อาจมีคีย์ stream/video, public key และลายเซ็น Playground ที่สร้างขึ้นของคุณ, ชื่อผู้เล่นที่สร้างขึ้นของคุณ, รายการเกมที่คุณพร้อมเล่น, คำเชิญและคำตอบคำเชิญ และการกระทำในเกม เช่น เดินหมากรุก

  การสร้างคำถาม HELP-A-FRIEND! Trivia อาจส่งส่วนที่เลือกจาก transcript ของ replay YouTube และตัวระบุเกมไปยัง backend ของ Playground backend ใช้ OpenAI เพื่อสร้างคำถาม trivia จากส่วนเหล่านั้น

  การสร้าง Replay Trivia อาจต้องใช้การตรวจสอบ Cloudflare Turnstile ที่ `https://playground.chatenhancer.com` Cloudflare อาจได้รับข้อมูลตรวจสอบทั่วไป เช่น IP address, user agent และผล challenge

  Playground ไม่ส่งข้อความแชทสด ชื่อแสดง YouTube ของคุณ URL avatar YouTube ของคุณ cookies ของ YouTube หรือ credentials ของ YouTube ไปยัง backend ของ Playground

  เช่นเดียวกับบริการเว็บอื่น backend ของ Playground อาจได้รับ metadata การเชื่อมต่อทั่วไป เช่น IP address และ user agent จากเบราว์เซอร์หรือผู้ให้บริการเครือข่าย

## การควบคุมข้อมูล

คุณสามารถล้างข้อมูลส่วนขยายจาก popup ของส่วนขยายโดยใช้ปุ่มรีเซ็ต การดำเนินการนี้จะล้างข้อมูลส่วนขยายในเครื่องและการตั้งค่าส่วนขยายที่ซิงค์ไว้ จากนั้นคืนค่าการตั้งค่าเริ่มต้น

คุณยังสามารถนำส่วนขยายออกจากเบราว์เซอร์ได้ ขึ้นอยู่กับเบราว์เซอร์ การนำส่วนขยายออกอาจลบ storage ในเครื่องของส่วนขยายนั้นด้วย

## สิ่งที่ไม่ได้เก็บรวบรวม

ส่วนขยายไม่เรียกใช้การวิเคราะห์

ส่วนขยายไม่เก็บประวัติการท่องเว็บ

ส่วนขยายไม่ขายข้อมูลผู้ใช้

ยกเว้นเกม Playground แบบเลือกเข้าร่วมที่อธิบายข้างต้น ส่วนขยายไม่ส่งข้อมูลไปยังเซิร์ฟเวอร์ที่ส่วนขยายเป็นเจ้าของ

ส่วนขยายไม่เก็บข้อความโปรไฟล์ล่าสุดหรือผลการแปลหลังจากหน้าแชทสด unload

Chat Enhancer for YouTube ไม่ได้เป็นพันธมิตรกับ YouTube หรือ Google
