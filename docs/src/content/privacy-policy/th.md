# นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: 17 มิถุนายน 2026

คำแปลนี้จัดทำเพื่อความสะดวก เวอร์ชันภาษาอังกฤษที่ /privacy/ เป็นแหล่งอ้างอิงหลัก

ฟีเจอร์ส่วนใหญ่ของ Chat Enhancer for YouTube ทำงานในเครื่องภายในเบราว์เซอร์ การแปลปิดไว้ตามค่าเริ่มต้น; เมื่อเปิดใช้งาน ข้อความที่แปลจะถูกส่งไปยัง Google Translate เกม Playground ปิดไว้ตามค่าเริ่มต้น หากเปิดและใช้งาน สถานะการเล่นเกม คำเชิญ และการกระทำในเกมจะถูกส่งไปยัง backend ของ Chat Enhancer Playground ภายใต้ชื่อผู้เล่นที่สร้างขึ้น

## ข้อมูลที่เก็บในเบราว์เซอร์

ส่วนขยายใช้พื้นที่เก็บข้อมูลของเบราว์เซอร์สำหรับการตั้งค่า ข้อมูล Inbox คีย์เวิร์ดที่ติดตาม อีโมจิที่ใช้บ่อย ผู้ใช้ที่บันทึกไว้ แบบร่างแชทที่ยังไม่ส่งแยกตามสตรีม และสถานะแท็บ live chat หากใช้ Playground ตัวตนแบบนามแฝงของ Playground จะถูกเก็บไว้ในเครื่อง ข้อความโปรไฟล์ล่าสุดและผลการแปลจะอยู่ในหน่วยความจำของหน้า live chat ปัจจุบันเท่านั้น และจะถูกลบเมื่อออกจากหน้า

## ข้อมูลที่ส่งออกนอกเบราว์เซอร์

ข้อมูลอาจถูกส่งออกนอกเบราว์เซอร์เฉพาะเมื่อเปิดใช้การแปลหรือ Playground คำขอแปลจะส่งไปยัง Google Translate และไม่รวมคุกกี้หรือข้อมูลรับรอง YouTube ของคุณ Playground อาจได้รับคีย์สตรีม/วิดีโอ public key และลายเซ็นที่สร้างขึ้น ชื่อผู้เล่นที่สร้างขึ้น เกมที่พร้อมเล่น คำเชิญ และการกระทำในเกม เช่น เดินหมากรุก HELP-A-FRIEND! Trivia อาจส่งข้อความถอดเสียง replay ของ YouTube บางส่วนและตัวระบุเกมไปยัง backend ของ Playground ซึ่งใช้ OpenAI เพื่อสร้างคำถาม Replay Trivia อาจต้องใช้การยืนยัน Cloudflare Turnstile และ Cloudflare อาจได้รับข้อมูลการยืนยันตามปกติ เช่น ที่อยู่ IP, user agent และผล challenge

## การควบคุมและข้อจำกัด

คุณสามารถล้างข้อมูลส่วนขยายจาก popup ด้วยปุ่มรีเซ็ต หรือเอาส่วนขยายออกจากเบราว์เซอร์ ส่วนขยายไม่เรียกใช้ analytics ไม่เก็บประวัติการท่องเว็บ และไม่ขายข้อมูลผู้ใช้ ยกเว้นเกม Playground แบบเลือกใช้ที่อธิบายไว้ข้างต้น ส่วนขยายจะไม่ส่งข้อมูลไปยังเซิร์ฟเวอร์ของส่วนขยาย Chat Enhancer for YouTube ไม่ได้มีความเกี่ยวข้องกับ YouTube หรือ Google
