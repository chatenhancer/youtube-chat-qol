# 개인정보 보호

마지막 업데이트: 2026년 6월 21일

Chat Enhancer for YouTube는 YouTube 실시간 채팅용 브라우저 확장 프로그램입니다. YouTube 채팅을 대체하거나 분석 데이터를 수집하지 않고 작은 채팅 기능을 추가하도록 설계되었습니다.

요약:

- 대부분의 확장 프로그램 기능은 브라우저에서 로컬로 실행됩니다.
- 번역은 기본적으로 꺼져 있습니다.
- 번역을 켜면 번역되는 텍스트가 Google Translate로 전송됩니다.
- Playground 게임은 기본적으로 꺼져 있습니다. Playground를 켜고 사용하면 게임 접속 상태, 초대, 게임 동작이 생성된 플레이어 이름으로 Chat Enhancer Playground 게임 서버에 전송됩니다.
- 확장 프로그램은 분석을 실행하지 않고, 데이터를 판매하지 않으며, 탐색 기록을 수집하지 않습니다.

## 확장 프로그램이 실행되는 위치

확장 프로그램은 확장 프로그램이 접근할 수 있도록 허용된 YouTube 실시간 채팅 및 실시간 채팅 리플레이 페이지에서만 실행됩니다.

확장 프로그램은 자체 설정과 데이터를 브라우저에 저장하기 위한 권한을 사용합니다. 또한 기능 작동에 필요한 특정 웹사이트에 대한 접근 권한을 사용합니다. 여기에는 YouTube 실시간 채팅 페이지, Google Translate 번역 서비스, 선택형 Chat Enhancer Playground 게임 서버가 포함됩니다.

확장 프로그램은 일반적인 탐색 기록, 탭 읽기, 스크립팅 또는 웹 탐색 권한을 요청하지 않습니다.

## 브라우저에 저장되는 데이터

확장 프로그램은 페이지를 새로고침해도 기능이 작동하도록 일부 데이터를 저장합니다.

이 섹션에 나열된 데이터는 확장 프로그램이 사용자의 브라우저 프로필에 저장합니다. 아래의 "브라우저 밖으로 전송되는 데이터" 섹션에도 나열되어 있지 않은 한 Chat Enhancer로 전송되지 않습니다.

- **설정:** 브라우저의 동기화된 확장 프로그램 저장소(`chrome.storage.sync`)를 사용해 저장됩니다. 브라우저 설정에 따라 브라우저가 로그인된 본인 브라우저 설치 간에 이러한 확장 프로그램 설정을 동기화할 수 있습니다.

- **Inbox 데이터:** 로컬 확장 프로그램 저장소(`chrome.storage.local`)를 사용해 저장됩니다. 여기에는 감시 키워드와 스트림 또는 리플레이당 최대 100개의 inbox 기록이 포함됩니다. Inbox 기록에는 메시지 텍스트, 작성자 이름, 타임스탬프, 저장된 메시지가 어디에서 왔는지 보여 주는 데 필요한 기본 YouTube 메시지 세부정보, 매칭 세부정보, 저장된 메시지를 올바르게 표시하는 데 필요한 이모지 또는 이미지 정보가 포함될 수 있습니다.

- **자주 사용하는 이모지 데이터:** 로컬 확장 프로그램 저장소(`chrome.storage.local`)를 사용해 저장됩니다. 여기에는 자주 사용하는 이모지 행을 만드는 데 사용되는 로컬 사용 횟수와 이모지 표시 정보가 포함됩니다.

- **북마크 데이터:** 로컬 확장 프로그램 저장소(`chrome.storage.local`)에 저장됩니다. 저장한 메시지 텍스트와 이모지 표시 정보, 작성자 이름, 아바타 URL과 사용 가능한 경우 채널 ID, 메시지 및 저장 시간, 스트림 제목과 URL이 포함될 수 있습니다. 북마크는 현재 브라우저 프로필에서 스트림 간에 계속 사용할 수 있습니다.

- **아바타 링 데이터:** 로컬 확장 프로그램 저장소(`chrome.storage.local`)에 저장됩니다. 최근 메시지 프로필에서 명시적으로 링을 추가한 사용자의 작성자 이름, 사용 가능한 경우 채널 ID, 링을 추가한 시간이 포함됩니다. 선택은 현재 브라우저 프로필에서 스트림 간에 계속 사용할 수 있으며 일치하는 아바타를 꾸미는 데만 사용됩니다. 사용자가 온라인인지 확인하지 않습니다.

- **보내지 않은 채팅 초안:** 스트림별로 로컬 확장 프로그램 저장소(`chrome.storage.local`)를 사용해 저장됩니다. 페이지 새로고침 후 복원됩니다. 초안은 채팅 입력이 지워지거나, 메시지가 전송되거나, 확장 프로그램 데이터가 재설정될 때 제거됩니다.

- **Playground 신원 데이터:** Playground를 사용하는 경우 로컬 확장 프로그램 저장소(`chrome.storage.local`)를 사용해 저장됩니다. 이는 Playground에 다시 연결할 때 같은 브라우저 설치를 인식하는 데 사용되는 무작위로 생성된 로컬 Playground 신원입니다. 이는 YouTube 신원이 아닙니다.

- **최근 프로필 메시지, 명령 상태, 번역 결과:** 현재 실시간 채팅 페이지의 메모리에만 보관됩니다. 채팅 페이지를 나가거나 새로고침하면 지워집니다.

## 브라우저 밖으로 전송되는 데이터

채팅 번역, 초안 번역, Playground 게임은 기본적으로 꺼져 있습니다.

번역 또는 Playground 기능을 켜고 사용하면 데이터가 다음 서비스로 전송될 수 있습니다.

- **`https://translate.googleapis.com/translate_a/single`의 Google Translate**

  채팅 번역은 번역이 켜져 있는 동안 실시간 채팅에 표시되고 번역 대상인 채팅 메시지 텍스트를 전송합니다. 초안 번역은 채팅 상자에서 번역하도록 선택한 초안 텍스트를 전송합니다.

  번역 요청에는 번역할 텍스트와 대상 언어가 포함됩니다. 확장 프로그램은 번역 요청과 함께 YouTube 쿠키나 YouTube 자격 증명을 보내지 않습니다.

  `translate.googleapis.com`을 통한 Google Translate 접근은 비공식이며, 속도 제한을 받거나 변경되거나 사용할 수 없게 될 수 있습니다.

- <span id="playground"></span>**`https://playground.chatenhancer.com`의 Chat Enhancer Playground**

  Playground는 기본적으로 꺼져 있습니다. Playground를 켜고 게임 패널을 사용하면 확장 프로그램이 Chat Enhancer Playground 게임 서버에 연결되어 같은 스트림의 참여 사용자가 이용 가능 상태를 보고, 초대를 교환하고, 게임을 할 수 있습니다.

  Playground 메시지에는 YouTube 스트림 또는 동영상 식별자, 생성된 Playground 플레이어 신원, 생성된 플레이어 이름, 사용 가능한 게임 목록, 초대와 초대 응답, 체스 수 같은 게임 동작이 포함될 수 있습니다.

  Playground는 실시간 채팅 메시지 텍스트, YouTube 표시 이름, YouTube 아바타 URL, YouTube 쿠키 또는 YouTube 자격 증명을 Playground 게임 서버로 보내지 않습니다.

  별도로 HELP-A-FRIEND! Trivia 질문 생성은 선택된 공개 YouTube 동영상 자막 발췌문과 게임 식별자를 Playground 게임 서버로 보낼 수 있습니다. 이 발췌문은 실시간 채팅이 아니라 동영상 자막에서 가져옵니다. 서버는 OpenAI를 사용해 해당 발췌문에서 trivia 질문을 생성합니다.

  Replay Trivia 생성에는 `https://playground.chatenhancer.com`에서 Cloudflare Turnstile 확인이 필요할 수 있습니다. Cloudflare는 IP 주소, 브라우저 및 기기 정보, challenge 결과 같은 일반적인 확인 데이터를 받을 수 있습니다.

  모든 웹 서비스와 마찬가지로 Playground 게임 서버는 브라우저 또는 네트워크 제공자로부터 IP 주소와 브라우저/기기 정보 같은 일반 연결 정보를 받을 수 있습니다.

## 데이터 제어

확장 프로그램 팝업의 재설정 버튼을 사용해 확장 프로그램 데이터를 지울 수 있습니다. 이렇게 하면 로컬 확장 프로그램 데이터와 동기화된 확장 프로그램 설정이 지워지고 기본 설정이 복원됩니다.

브라우저에서 확장 프로그램을 제거할 수도 있습니다. 브라우저에 따라 확장 프로그램을 제거하면 로컬 확장 프로그램 저장소도 제거될 수 있습니다.

## Chat Enhancer가 하지 않는 일

확장 프로그램은 분석을 실행하지 않습니다.

확장 프로그램은 탐색 기록을 수집하지 않습니다.

확장 프로그램은 사용자 데이터를 판매하지 않습니다.

위에 설명된 선택형 Playground 기능을 제외하고, 확장 프로그램은 Chat Enhancer 서버로 데이터를 보내지 않습니다.

확장 프로그램은 실시간 채팅 페이지를 나가거나 새로고침한 후 최근 프로필 메시지나 번역 결과를 저장하지 않습니다.

Chat Enhancer for YouTube는 YouTube 또는 Google과 제휴하지 않습니다.
