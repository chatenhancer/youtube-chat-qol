# 개인정보 처리방침

최종 업데이트: 2026년 6월 17일

이 번역은 편의를 위해 제공됩니다. /privacy/의 영어 버전이 기본 기준입니다.

Chat Enhancer for YouTube의 대부분 기능은 브라우저에서 로컬로 실행됩니다. 번역은 기본적으로 꺼져 있으며, 켜면 번역되는 텍스트가 Google Translate로 전송됩니다. Playground 게임은 기본적으로 꺼져 있습니다. 이를 켜고 사용하면 게임 접속 상태, 초대, 게임 동작이 생성된 플레이어 이름으로 Chat Enhancer Playground 백엔드에 전송됩니다.

## 브라우저에 저장되는 데이터

확장 프로그램은 설정, Inbox 데이터, 감시 키워드, 자주 쓰는 이모지, 북마크한 사용자, 스트림별 미전송 채팅 초안, 라이브 채팅 탭 상태를 위해 브라우저 저장소를 사용합니다. Playground를 사용하면 익명에 가까운 Playground ID가 로컬에 저장됩니다. 최근 프로필 메시지와 번역 결과는 현재 라이브 채팅 페이지의 메모리에만 보관되며 페이지를 떠나면 삭제됩니다.

## 브라우저 밖으로 전송되는 데이터

데이터는 번역 또는 Playground가 활성화된 경우에만 브라우저 밖으로 전송될 수 있습니다. 번역 요청은 Google Translate로 전송되며 YouTube 쿠키나 인증 정보는 포함하지 않습니다. Playground는 스트림/동영상 키, 생성된 공개 키와 서명, 생성된 플레이어 이름, 사용 가능한 게임, 초대, 체스 수 같은 게임 동작을 받을 수 있습니다. HELP-A-FRIEND! Trivia는 선택된 YouTube 리플레이 자막 일부와 게임 식별자를 Playground 백엔드로 보낼 수 있으며, 백엔드는 OpenAI를 사용해 질문을 생성합니다. Replay Trivia는 Cloudflare Turnstile 확인이 필요할 수 있고, Cloudflare는 IP 주소, 사용자 에이전트, 챌린지 결과 같은 일반 확인 데이터를 받을 수 있습니다.

## 관리 및 제한

확장 프로그램 팝업의 재설정 버튼으로 확장 프로그램 데이터를 지우거나 브라우저에서 확장 프로그램을 제거할 수 있습니다. 확장 프로그램은 분석을 실행하지 않고, 브라우징 기록을 수집하지 않으며, 사용자 데이터를 판매하지 않습니다. 위에서 설명한 선택형 Playground 게임을 제외하고 확장 프로그램 소유 서버로 데이터를 보내지 않습니다. Chat Enhancer for YouTube는 YouTube 또는 Google과 제휴하지 않았습니다.

개인정보 관련 질문은 https://www.chatenhancer.com의 이메일 링크를 사용하세요.
