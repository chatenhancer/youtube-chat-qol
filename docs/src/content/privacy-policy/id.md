# Kebijakan privasi

Terakhir diperbarui: 17 Juni 2026

Terjemahan ini disediakan untuk kemudahan. Versi bahasa Inggris di /privacy/ adalah rujukan utama.

Sebagian besar fitur Chat Enhancer for YouTube berjalan secara lokal di browser. Terjemahan nonaktif secara default; saat diaktifkan, teks yang diterjemahkan dikirim ke Google Translate. Game Playground nonaktif secara default. Jika Anda mengaktifkan dan menggunakannya, status ketersediaan game, undangan, dan aksi game dikirim ke backend Chat Enhancer Playground dengan nama pemain yang dibuat otomatis.

## Data yang disimpan di browser

Ekstensi menggunakan penyimpanan browser untuk setelan, data Inbox, kata kunci yang dipantau, emoji yang sering digunakan, pengguna yang ditandai, draf chat yang belum dikirim per stream, dan status tab live chat. Identitas Playground pseudonim disimpan secara lokal jika Playground digunakan. Pesan profil terbaru dan hasil terjemahan hanya disimpan di memori halaman live chat saat ini dan dihapus saat halaman ditutup.

## Data yang dikirim ke luar browser

Data dapat dikirim ke luar browser hanya saat terjemahan atau Playground diaktifkan. Permintaan terjemahan dikirim ke Google Translate dan tidak menyertakan cookie atau kredensial YouTube Anda. Playground dapat menerima kunci stream/video, kunci publik dan tanda tangan yang dibuat, nama pemain yang dibuat, daftar game yang tersedia, undangan, dan aksi game seperti langkah catur. HELP-A-FRIEND! Trivia dapat mengirim cuplikan transkrip replay YouTube terpilih dan pengenal game ke backend Playground, yang menggunakan OpenAI untuk membuat pertanyaan. Replay Trivia dapat memerlukan verifikasi Cloudflare Turnstile, dan Cloudflare dapat menerima data verifikasi normal seperti alamat IP, user agent, dan hasil tantangan.

## Kontrol dan batasan

Anda dapat menghapus data ekstensi dari popup dengan tombol reset, atau menghapus ekstensi dari browser. Ekstensi tidak menjalankan analitik, tidak mengumpulkan riwayat penjelajahan, dan tidak menjual data pengguna. Kecuali game Playground opt-in yang dijelaskan di atas, ekstensi tidak mengirim data ke server milik ekstensi. Chat Enhancer for YouTube tidak berafiliasi dengan YouTube atau Google.

Untuk pertanyaan privasi, gunakan tautan email di https://www.chatenhancer.com.
