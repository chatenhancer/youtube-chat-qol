## ทำไมต้อง Chat Enhancer?

YouTube live chat เก่งมากเรื่องความเร็ว แต่ไม่ค่อยเก่งเท่าไรถ้าคุณอยากตามบทสนทนาให้ทันจริง ๆ

ผมสร้าง Chat Enhancer หลังจากใช้เวลามากเกินไปใน livestream ที่เปิดยาวนานมาก มีคนจากทั่วโลกคุยกัน ข้อความไหลเร็ว และเมื่อเวลาผ่านไปมันเริ่มรู้สึกเหมือนชุมชนเล็ก ๆ ชั่วคราว สิ่งนี้ยังทำให้เห็นปัญหาชัดเจน: mentions พลาดได้ง่าย บทสนทนาหายไปทันที กำแพงภาษาขวางทาง และการดูบริบทพื้นฐานเกี่ยวกับคนอื่นใน chat ทำได้ยากกว่าที่ควร

ดังนั้นผมจึงทำส่วนขยายสำหรับสิ่งที่ผมอยากให้ YouTube live chat มีอยู่แล้ว

Chat Enhancer ถูกตั้งใจให้วางตัวเงียบ ๆ อยู่ใน YouTube ไม่ใช่มาแทนที่มัน Chat ยังคงให้ความรู้สึกเหมือน YouTube chat เพียงแต่มีเครื่องมือเพิ่มขึ้นเล็กน้อยในจุดที่มีประโยชน์

## การแปลแบบสด

ภาษาเป็นปัญหาแรกที่ผมเจอซ้ำ ๆ Chat มีข้อความภาษาเกาหลีและจีนจำนวนมาก การแปลทั้งหน้าทำงานได้ในทางเทคนิค แต่ก็ทำให้ chat รู้สึกแปลกและตามยากขึ้น

Chat Enhancer ให้คุณแปลข้อความ live chat ได้โดยตรง พร้อมรักษา emoji และ mention ไว้เหมือนเดิม คุณสามารถ hover ข้อความที่แปลแล้วเพื่อดูข้อความต้นฉบับและภาษาที่ตรวจพบ

คุณยังแปลข้อความที่กำลังพิมพ์ก่อนส่งได้ด้วย ทำให้การมีส่วนร่วมใน chat หลายภาษาง่ายขึ้นมาก แทนที่จะได้แต่ดูอยู่ข้าง ๆ

การแปลใช้ Google Translate จึงไม่สมบูรณ์แบบ แต่ในทางปฏิบัติยังดีพอที่จะช่วยให้ติดตาม chat หลายภาษาที่คึกคักได้ง่ายขึ้น

## Inbox

YouTube live chat ไม่แจ้งเตือนเมื่อมีคน mention คุณ

นั่นไม่เป็นไรถ้าคุณจ้อง chat ทุกวินาที แต่ไม่ค่อยโอเคถ้าคุณเปิด stream ทิ้งไว้เบื้องหลัง ไปชงกาแฟ หรือเดินออกไปสักหนึ่งนาที

Chat Enhancer เพิ่ม inbox สำหรับ mentions และ keywords ที่ติดตาม เพื่อให้คุณตามข้อความที่สำคัญได้โดยไม่ต้องเลื่อนดูทุกอย่างเอง

## บริบทผู้ใช้ที่ดีขึ้น

Live chat เคลื่อนเร็ว เมื่อมีคนตอบคุณ ไม่ได้ชัดเจนเสมอไปว่าเขาเป็นใครหรือเมื่อไม่กี่นาทีก่อนเขาพูดถึงเรื่องอะไร

ด้วย Chat Enhancer การคลิก avatar ของผู้ใช้จะเปิด panel โปรไฟล์พร้อมข้อความล่าสุดของคนนั้น คุณยังเปิดช่อง YouTube ของเขาใน popup ด้านข้างและ bookmark ผู้ใช้ที่อยากจำได้ในภายหลังได้

เป็นเรื่องเล็ก ๆ แต่ทำให้คนที่คุยประจำดูไม่เหมือนชื่อสุ่ม ๆ ที่ลอยผ่านไป

## Emoji ที่ใช้บ่อยที่สุด

ใช้ emoji เดิม ๆ ตลอด การต้องเลื่อนผ่านตัวเลือก emoji ทั้งหมดทุกครั้งเริ่มน่ารำคาญอย่างรวดเร็ว

Chat Enhancer เพิ่มแถวสำหรับ emoji ที่คุณใช้บ่อยที่สุด เพื่อให้ emoji ที่คุณใช้จริง ๆ เข้าถึงง่ายขึ้น

## การปรับปรุงเล็ก ๆ อื่น ๆ

ยังมีฟีเจอร์เล็ก ๆ เพิ่มเติม เช่น mention และ quote อย่างรวดเร็ว, bookmark ผู้ใช้, chat commands, draft recovery และรายละเอียดเล็ก ๆ ที่ทำให้ live chat ไม่รู้สึกพื้นฐานเกินไป
