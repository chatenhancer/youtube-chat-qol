## ¿Por qué Chat Enhancer?

El chat en directo de YouTube es muy bueno siendo rápido. Es menos bueno cuando quieres seguir conversaciones de verdad.

Creé Chat Enhancer después de pasar demasiado tiempo en un directo que llevaba mucho tiempo activo. Había gente de todo el mundo, los mensajes iban muy rápido y, con el tiempo, aquello empezó a sentirse como una pequeña comunidad temporal. También dejó claras varias cosas: las menciones se pierden con facilidad, las conversaciones desaparecen al instante, las barreras de idioma molestan y obtener contexto básico sobre otras personas del chat es más difícil de lo que debería.

Así que hice una extensión para las cosas que siempre acababa deseando que YouTube live chat ya tuviera.

Chat Enhancer está pensado para quedarse discretamente dentro de YouTube, no para reemplazarlo. El chat sigue sintiéndose como el chat de YouTube; solo tiene unas cuantas herramientas más donde resultan útiles.

## Traducción en directo

El idioma fue el primer problema con el que me seguí encontrando. El chat tenía muchos mensajes en coreano y chino. Traducir toda la página funcionaba técnicamente, pero también hacía que el chat se sintiera raro y más difícil de seguir.

Chat Enhancer te permite traducir mensajes del chat en directo directamente, manteniendo intactos emojis y menciones. Puedes pasar el cursor sobre un mensaje traducido para ver el texto original y el idioma detectado.

También puedes traducir lo que estás escribiendo antes de enviarlo, lo que hace mucho más fácil participar de verdad en un chat multilingüe en lugar de mirar desde la barrera.

La traducción usa Google Translate, así que no será perfecta. En la práctica, sigue siendo lo bastante buena como para hacer que un chat multilingüe con mucho movimiento sea más fácil de seguir.

## Inbox

YouTube live chat no te avisa cuando alguien te menciona.

Eso está bien si miras el chat cada segundo. Está menos bien si dejas el directo de fondo, preparas café o te alejas un minuto.

Chat Enhancer añade un inbox para menciones y palabras clave vigiladas, para que puedas ponerte al día con los mensajes que te importan sin tener que revisar todo manualmente.

## Mejor contexto de usuarios

Los chats en directo se mueven rápido. Cuando alguien te responde, no siempre es obvio quién es ni de qué hablaba hace unos minutos.

Con Chat Enhancer, al hacer clic en el avatar de un usuario se abre un panel de perfil con mensajes recientes de esa persona. También puedes abrir su canal de YouTube en una ventana lateral y guardar usuarios que quieras reconocer más adelante.

Es algo pequeño, pero hace que quienes participan a menudo se sientan menos como nombres aleatorios que pasan volando.

## Emojis más usados

Uso los mismos emojis constantemente. Tener que recorrer todo el selector cada vez se volvió molesto enseguida.

Chat Enhancer añade una fila para tus emojis más usados, para que los que realmente usas estén más a mano.

## Otras pequeñas mejoras

También incluye algunas mejoras más, como menciones y citas rápidas, usuarios guardados, comandos de chat, recuperación de borradores y otros detalles pensados para que el chat en directo se sienta un poco menos básico.

Nada de esto intenta convertir el chat en directo de YouTube en una app distinta. Solo hace que el chat existente sea más fácil de usar.

Chat Enhancer for YouTube es gratis, de código abierto, no requiere cuenta y no usa analíticas.
