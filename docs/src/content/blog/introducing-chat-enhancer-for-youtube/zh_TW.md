## 為什麼做 Chat Enhancer？

YouTube 直播聊天室的節奏很快，這正是它的魅力。但如果你想真正跟上對話，它就沒有那麼友善了。

我開始做 Chat Enhancer，是因為自己在一場長時間直播裡待了太久。來自世界各地的人在同一個聊天室裡說話，訊息跑得飛快，時間一久，它像是一個臨時聚在一起的小社群。問題也跟著變得很明顯：被人提到時很容易錯過，剛聊過的話題馬上就被沖走，語言隔閡會擋住交流，想知道正在說話的人是誰也比想像中費力。

於是我做了這個擴充功能，把我一直希望 YouTube 直播聊天室自帶的那些小功能補上。

Chat Enhancer 只想安靜地嵌在 YouTube 裡，而不是取代 YouTube。聊天室看起來仍然是 YouTube 的聊天室，只是在需要的地方多了幾個順手的工具。

## 即時翻譯

語言是我最早反覆遇到的問題。聊天室裡常常混著韓文和中文訊息。整頁翻譯在技術上能用，但會把聊天室介面弄得很奇怪，讀起來反而更累。

Chat Enhancer 可以直接翻譯直播聊天室訊息，同時保留表情符號和提及。把游標停在譯文上，還能查看原文和偵測到的語言。

你也可以在送出前翻譯自己正在輸入的內容。這樣一來，多語言聊天就不只是旁觀，你也更容易真正加入進去。

翻譯使用 Google Translate，所以不會完美。但實際用下來，它已經足夠讓繁忙的多語聊天室更容易跟上。

## Inbox

YouTube 直播聊天室不會在有人提到你時提醒你。

如果你每秒都盯著聊天室，這當然沒問題。可如果你把直播放在背景、去泡咖啡，或只是離開一分鐘，就很容易錯過。

Chat Enhancer 會為提及和追蹤關鍵字提供 Inbox，讓你不用捲完整個聊天室，也能回頭看和自己有關的重要訊息。

## 更清楚的使用者脈絡

直播聊天室刷得很快。有人回覆你時，你不一定馬上看得出對方是誰，也不一定記得幾分鐘前他們在說什麼。

使用 Chat Enhancer 時，點擊使用者頭像會開啟個人資料面板，顯示這個使用者最近發過的訊息。你也可以在側邊小視窗開啟對方的 YouTube 頻道，並標記那些之後想認得的使用者。

這只是個小功能，卻能讓常出現的聊天者不再像一串飛快掠過的隨機名字。

## 最常使用的表情符號

我自己總是用那幾個表情符號。每次都要在完整表情選擇器裡找，久了很煩。

Chat Enhancer 會加上一排最常使用的表情符號，讓你真正常用的那些更容易點到。

## 其他小改進

還有一些別的順手改進：快速提及和引用、使用者標記、聊天指令、草稿復原，以及其他讓直播聊天室不那麼陽春的小細節。

這些功能都不是要把 YouTube 直播聊天室變成另一個應用程式，只是讓現有聊天室更好用一點。
