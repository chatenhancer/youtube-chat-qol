## 為什麼做 Chat Enhancer？

YouTube 直播聊天室很擅長快速流動。但如果你想真正跟上對話，它就沒那麼擅長了。

我是在一個長時間運行的 livestream 裡待了太久之後做出 Chat Enhancer 的。那裡有來自世界各地的人聊天，訊息移動得很快，久而久之，它開始像一個小型的臨時社群。同時幾個問題也變得很明顯：提及很容易錯過，對話會瞬間消失，語言隔閡會造成阻礙，而想取得聊天室中其他人的基本脈絡也比應該有的更困難。

所以我做了一個擴充功能，補上那些我一直希望 YouTube 直播聊天室本來就有的東西。

Chat Enhancer 的設計目標是像 YouTube 的一部分，而不是取代它。它保留原生直播聊天體驗，只在上面加上一層小小的實用改進。

## 即時翻譯

我最先遇到的問題之一是語言。當時的聊天室裡有很多韓文和中文訊息。對整個頁面使用 Google Translate 技術上可行，但會讓聊天室感覺奇怪，也更難跟上。

Chat Enhancer 可以直接翻譯直播聊天室訊息，同時保留 emoji 和提及。你可以將游標停在翻譯後的訊息上，查看原文和偵測到的語言。

你也可以在送出前翻譯正在輸入的內容。這讓你更容易真正參與多語言聊天，而不只是從旁觀看。

翻譯透過 Google Translate 處理，所以並不完美，但已經足夠實用，可以帶來真實差異。

## Inbox

YouTube 直播聊天室不會在有人提到你時通知你。

如果你每一秒都盯著聊天室，這沒問題。但如果你把直播放在背景、去泡咖啡，或短暫假裝自己有生活，那就比較不理想。

Chat Enhancer 為提及和追蹤關鍵字加入 Inbox，讓你可以回頭查看重要訊息，不必手動捲過所有內容。

## 更好的使用者脈絡

直播聊天室移動很快，不一定容易知道某個人是誰，或他剛才說了什麼。

使用 Chat Enhancer 時，點擊使用者頭像會開啟一個個人資料面板，顯示該使用者最近的訊息。你也可以在側邊 popup 開啟他的 YouTube 頻道，並 bookmark 之後想認出的使用者。

這是小功能，但會讓經常出現的聊天者感覺不再那麼匿名。

## 最常使用的 emoji

我一直使用同樣的 emoji。每次都要捲完整個 emoji 選擇器很快就變得麻煩。

Chat Enhancer 會加入一排最常使用的 emoji，讓你真正會用的 emoji 更容易取得。

## 其他小改進

還有一些其他便利功能，例如快速提及和引用、使用者 bookmark、聊天指令、草稿復原，以及讓直播聊天不那麼陽春的其他小細節。

這些都不是要重新發明 YouTube 直播聊天室。只是讓它更好用。

Chat Enhancer for YouTube 免費、開放原始碼、不需要帳號，也不執行 analytics。
