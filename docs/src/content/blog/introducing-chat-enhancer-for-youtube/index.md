## Why Chat Enhancer?

YouTube live chat is great at being fast. It is less great at being a place where you can actually keep track of conversations.

I built Chat Enhancer after spending far too much time in a long-running livestream. The stream had people chatting from all over the world, messages moved quickly, and over time it started to feel like a small temporary community. It also made a few things painfully obvious: mentions are easy to miss, conversations disappear instantly, language barriers get in the way, and basic context about other people in chat is harder to get than it should be.

So I made an extension for the things I kept wishing YouTube live chat already had.

Chat Enhancer is designed to feel like part of YouTube, rather than a replacement for it. It keeps the native live chat experience, but adds a small layer of quality-of-life features on top.

## Live translation

One of the first problems I ran into was language. The chat I was in had a lot of Korean and Chinese messages, and using Google Translate on the whole page technically worked, but it also made the chat feel strange and harder to follow.

Chat Enhancer lets you translate live chat messages directly, while keeping emojis and mentions intact. You can hover over a translated message to see the original text and detected language.

You can also translate what you are typing before sending it, which makes it much easier to actually participate in a multilingual chat instead of just watching from the sidelines.

Translation is handled through Google Translate, so it is not perfect, but it is useful enough to make a real difference.

## Inbox

YouTube live chat does not notify you when someone mentions you.

That is fine if you are staring at the chat every second. It is less fine if you leave the stream in the background, go make coffee, or briefly pretend to have a life.

Chat Enhancer adds an inbox for mentions and watched keywords, so you can catch up on messages that matter to you without having to scroll through everything manually.

## Better user context

Live chats move quickly, and it is not always easy to know who someone is or what they have been saying.

With Chat Enhancer, clicking a user’s avatar opens a profile panel with recent messages from that user. You can also open their YouTube channel in a side popup and bookmark users you want to recognize later.

It is a small thing, but it makes recurring chatters feel a lot less anonymous.

## Most-used emojis

I use the same emojis constantly. Scrolling through the full emoji picker every time got old fast.

Chat Enhancer adds a row for your most-used emojis, so the ones you actually use are easier to reach.

## Other small improvements

There are a few more quality-of-life features included, like quick mentions and quotes, user bookmarks, chat commands, draft recovery, and other small touches designed to make live chat feel a little less barebones.

None of it is meant to reinvent YouTube live chat. It is just meant to make it more usable.

Chat Enhancer for YouTube is free, open-source, requires no account, and does not run analytics.

You can learn more at [chatenhancer.com](https://www.chatenhancer.com/#features)
