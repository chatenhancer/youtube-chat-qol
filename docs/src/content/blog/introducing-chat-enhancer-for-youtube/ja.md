## なぜ Chat Enhancer なのか

YouTube のライブチャットは、とにかく速いという点では優れています。ただ、会話をきちんと追いかける場所としてはあまり得意ではありません。

Chat Enhancer は、長時間続く livestream にあまりにも長く居続けたあとに作りました。世界中の人が話していて、メッセージはすぐに流れ、時間がたつにつれて小さな一時的なコミュニティのように感じられてきました。同時に、いくつかの問題もはっきり見えてきました。メンションは簡単に見逃す、会話はすぐに消える、言語の壁が邪魔をする、そしてチャット内の他の人について基本的な文脈を得るのが必要以上に難しい、ということです。

そこで、YouTube ライブチャットに最初からあってほしかった機能をまとめた拡張機能を作りました。

Chat Enhancer は、YouTube を置き換えるのではなく、YouTube の中に静かに収まることを目指しています。チャットは今までどおり YouTube チャットらしく感じられ、必要な場所にいくつかのツールが増えるだけです。

## ライブ翻訳

最初に何度もぶつかった問題は言語でした。チャットには韓国語や中国語のメッセージが多くありました。ページ全体を翻訳すること自体はできましたが、チャットが不自然になり、かえって追いにくくなりました。

Chat Enhancer では、絵文字やメンションを保ったままライブチャットのメッセージを直接翻訳できます。翻訳済みメッセージにカーソルを合わせると、元のテキストと検出された言語を確認できます。

送信前に、自分が入力している文章を翻訳することもできます。多言語のチャットをただ眺めるだけでなく、実際に参加しやすくなります。

翻訳には Google Translate を使うため完璧ではありません。実際には、忙しい多言語チャットを追いやすくするには十分です。

## Inbox

YouTube のライブチャットは、誰かがあなたにメンションしても通知してくれません。

毎秒チャットを見ているならそれでも問題ありません。ですが、配信をバックグラウンドに置いたり、コーヒーを入れたり、1分だけ席を外したりすると困ります。

Chat Enhancer は、メンションと監視キーワード用の Inbox を追加します。すべてを手動でスクロールしなくても、自分に関係のあるメッセージを後から確認できます。

## ユーザーの文脈を見やすく

ライブチャットは流れが速いです。誰かがあなたに返信したとき、その人が誰なのか、数分前に何について話していたのかが必ずしもすぐには分かりません。

Chat Enhancer では、ユーザーのアバターをクリックすると、その人の最近のメッセージを表示するプロフィールパネルが開きます。YouTube チャンネルをサイドポップアップで開いたり、後で見分けたいユーザーをブックマークしたりすることもできます。

小さな機能ですが、よく見かける参加者が通り過ぎるランダムな名前のようには感じにくくなります。

## よく使う絵文字

私はいつも同じ絵文字ばかり使います。毎回絵文字ピッカー全体をスクロールするのは、すぐに面倒になりました。

Chat Enhancer は、よく使う絵文字の行を追加します。実際によく使う絵文字にすぐ手が届きます。

## その他の小さな改善

ほかにも、素早いメンションと引用、ユーザーのブックマーク、チャットコマンド、下書きの復元など、ライブチャットを少しだけ使いやすくするための細かな改善が含まれています。

これらは YouTube ライブチャットを別のアプリに変えようとするものではありません。既存のチャットを使いやすくするだけです。

Chat Enhancer for YouTube は無料でオープンソース、アカウント不要で、analytics も実行しません。
