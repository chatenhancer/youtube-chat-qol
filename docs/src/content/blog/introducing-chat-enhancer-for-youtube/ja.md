## Chat Enhancer を作った理由

YouTube ライブチャットは、流れの速さが魅力です。でも、ひとつの会話を追い続ける場所としては、あまり親切ではありません。

Chat Enhancer を作り始めたのは、ある長時間のライブ配信に居続けていたときでした。世界中から人が集まり、メッセージはどんどん流れ、しばらくすると一時的な小さなコミュニティのように感じられてきました。同時に、弱いところもはっきり見えてきました。メンションは見逃しやすい。会話はすぐに流れて消える。言語の壁がある。そして、話している相手の基本的な文脈をつかむことさえ、思ったより難しいのです。

Chat Enhancer は、YouTube を置き換えるためのものではありません。YouTube の中に静かになじみ、チャットは今までどおり YouTube チャットのまま。必要な場所に、少しだけ便利なツールが増えるイメージです。

## ライブ翻訳

最初にぶつかった問題は言葉の壁でした。チャットには韓国語や中国語のメッセージが多くありましたが、ページ全体を翻訳すると YouTube の画面まで変わり、個々のメッセージを見分けにくくなりました。

Chat Enhancer では、絵文字やメンションを保ったまま、ライブチャットのメッセージをその場で翻訳できます。翻訳されたメッセージにカーソルを合わせると、元の文章と検出された言語も確認できます。

送信前に、自分が入力している文章を翻訳することもできます。多言語のチャットを横から眺めるだけでなく、実際に参加しやすくなります。

翻訳には Google Translate を使います。完璧ではありませんが、混み合った多言語チャットを追いやすくするには十分役立ちます。

## Inbox

YouTube ライブチャットでは、誰かにメンションされても通知は届きません。 Chat Enhancer は、メンションとウォッチ中のキーワードをまとめる Inbox を追加します。すべてのメッセージを手でさかのぼらなくても、自分に関係のある発言を後から確認できます。

## ユーザーの文脈を見やすく

Chat Enhancer では、ユーザーのアバターをクリックすると、その人の最近のメッセージを見られるプロフィールパネルが開きます。YouTube チャンネルをサイドポップアップで開いたり、後で見分けたいユーザーをブックマークしたりすることもできます。

## よく使う絵文字

Chat Enhancer は、よく使う絵文字を並べた行を追加します。実際に使う絵文字に、すぐ手が届くようになります。

## その他の小さな改善

ほかにも、すばやいメンションや引用、ユーザーのブックマーク、チャットコマンド、下書きの復元など、ライブチャットの使いにくさを少しずつ減らす改善を入れています。
