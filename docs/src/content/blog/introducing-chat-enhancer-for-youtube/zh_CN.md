## 为什么做 Chat Enhancer？

YouTube 直播聊天非常擅长“快”。但如果你想真正跟上对话，它就没那么擅长了。

我是在一个长期运行的 livestream 里待了太久之后做出 Chat Enhancer 的。那里有来自世界各地的人聊天，消息滚动得很快，时间久了，它开始像一个小小的临时社区。同时，一些问题也变得很明显：提及很容易错过，对话会瞬间消失，语言障碍会打断交流，想了解聊天里其他人的基本背景也比想象中更难。

所以我做了一个扩展，把那些我一直希望 YouTube 直播聊天自带的功能补上。

Chat Enhancer 的设计目标是像 YouTube 的一部分，而不是替代 YouTube。它保留原生直播聊天体验，只在上面加一层小而实用的改进。

## 实时翻译

我最先遇到的问题之一就是语言。我所在的聊天里有很多韩文和中文消息。对整个页面使用 Google Translate 技术上可以工作，但会让聊天变得奇怪，也更难跟上。

Chat Enhancer 可以直接翻译直播聊天消息，同时保持表情和提及不变。你可以将鼠标悬停在翻译后的消息上，查看原文和检测到的语言。

你还可以在发送前翻译自己正在输入的内容。这样你就更容易真正参与多语言聊天，而不是只在旁边看。

翻译由 Google Translate 处理，所以并不完美，但足够有用，能带来真实的改善。

## Inbox

YouTube 直播聊天不会在有人提到你时通知你。

如果你每一秒都盯着聊天，这没问题。但如果你把直播放在后台、去冲咖啡，或者短暂假装自己有生活，那就不太好了。

Chat Enhancer 为提及和关注的关键词添加了 Inbox，这样你可以回看重要消息，而不用手动滚动所有内容。

## 更好的用户上下文

直播聊天移动很快，不总是容易知道某个人是谁，或者他之前说过什么。

使用 Chat Enhancer 时，点击用户头像会打开一个资料面板，显示该用户最近的消息。你还可以在侧边弹窗打开他们的 YouTube 频道，并 bookmark 你以后想认出的用户。

这只是一个小功能，但会让经常出现的聊天者不那么匿名。

## 最常用表情

我总是使用同样的表情。每次都要滚完整个表情选择器很荒唐，因为显然我是一个情绪范围固定的简单人类。

Chat Enhancer 会添加一排最常用表情，让你真正会用的表情更容易找到。

## 其他小改进

还有一些其他便利功能，例如快速提及和引用、用户 bookmark、聊天命令、草稿恢复，以及一些让直播聊天不那么简陋的小细节。

这些功能都不是为了重新发明 YouTube 直播聊天。它们只是为了让它更好用。

Chat Enhancer for YouTube 免费、开源、不需要账号，也不运行 analytics。
