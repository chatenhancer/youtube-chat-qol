Playground est maintenant plus facile à lancer : vous pouvez jouer contre **Computer**.

Parfois, le chat est calme. Parfois, vos amis regardent un autre stream. Parfois, vous voulez simplement lancer une partie rapide sans attendre que quelqu’un accepte. Computer vous donne une option de plus.

## Comment ça marche

Ouvrez Playground depuis le panneau Jeux et cherchez Computer dans la liste des joueurs. Invitez Computer comme vous inviteriez un autre joueur. La partie démarre automatiquement, et vous pouvez jouer sans changer votre façon d’utiliser Playground.

Computer est disponible pour :

- **Chess**, pour une partie rapide en un contre un.
- **HELP-A-FRIEND! Trivia**, pour que les quiz de replay restent jouables même quand personne n’est libre.

## L’expérience reste Playground

Ce n’est pas un mode solo séparé ni un nouvel écran. Computer se comporte comme un adversaire Playground normal, donc le parcours reste familier : ouvrez le panneau, choisissez un jeu, envoyez une invitation et jouez.

Dans Chess, Computer joue ses coups après une courte pause pour que la partie ne paraisse pas instantanée ou trop mécanique.

Dans *HELP-A-FRIEND! Trivia*, Computer répond pendant le tour de question comme le ferait un autre joueur. Il n’est pas parfait, donc vous avez toujours une vraie chance de gagner.

## Pourquoi l’ajouter ?

Playground est plus amusant quand quelqu’un est là pour jouer avec vous, mais le chat en direct est imprévisible. Computer garde les jeux utilisables pendant les moments plus calmes, les streams tardifs, les replays ou les petites communautés où un autre utilisateur de Chat Enhancer n’est pas toujours disponible.

![Réglages Playground dans la fenêtre Chat Enhancer](./extension-settings.png)

Playground reste facultatif. Activez **Rejoindre Playground** dans les réglages de l’extension, ouvrez le panneau Jeux dans le chat et invitez Computer quand vous voulez lancer une partie.
