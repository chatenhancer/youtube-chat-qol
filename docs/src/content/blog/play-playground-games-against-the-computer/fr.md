Playground est maintenant plus facile à lancer : vous pouvez jouer contre **Computer**.

Parfois, le chat est calme. Parfois, vos amis regardent un autre stream. Parfois, vous voulez simplement lancer une partie rapide sans attendre que quelqu’un accepte. Computer vous donne une option de plus.

## Comment ça marche

Ouvrez Playground depuis le panneau Jeux et cherchez un joueur Computer dans la liste des joueurs. Invitez-en un comme vous inviteriez un autre joueur. La partie démarre automatiquement, et vous pouvez jouer sans changer votre façon d’utiliser Playground.

Les adversaires Computer sont disponibles pour :

- **Chess**, avec **Computer (Beginner)**, **Computer (Club)** et **Computer (Master)** pour choisir une partie plus légère, intermédiaire ou plus difficile.
- **HELP-A-FRIEND! Trivia**, pour que les quiz de replay restent jouables même quand personne n’est libre.

## Choisissez votre difficulté Chess

Chess propose maintenant trois adversaires Computer. Beginner est l’option la plus simple pour s’échauffer, Club est un adversaire intermédiaire plus régulier, et Master est l’option la plus difficile.

Aucun réglage séparé n’est nécessaire : choisissez le profil Computer voulu dans la liste des joueurs et envoyez une invitation.

## L’expérience reste Playground

Ce n’est pas un mode solo séparé ni un nouvel écran. Les joueurs Computer se comportent comme des adversaires Playground normaux, donc le parcours reste familier : ouvrez le panneau, choisissez un jeu, envoyez une invitation et jouez.

Dans Chess, Computer joue ses coups après une courte pause pour que la partie ne paraisse pas instantanée.

Dans *HELP-A-FRIEND! Trivia*, Computer répond pendant le tour de question comme le ferait un autre joueur. Il n’est pas parfait, donc vous avez toujours une vraie chance de gagner.

## Pourquoi l’ajouter ?

Playground est plus amusant quand quelqu’un est là pour jouer avec vous, mais le chat en direct est imprévisible. Computer garde les jeux utilisables pendant les moments plus calmes, les streams tardifs, les replays ou les petites communautés où un autre utilisateur de Chat Enhancer n’est pas toujours disponible.
