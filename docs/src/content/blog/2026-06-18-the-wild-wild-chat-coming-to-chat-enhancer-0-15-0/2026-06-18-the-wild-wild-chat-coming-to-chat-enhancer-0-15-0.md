The next Playground game is riding into live chat: **The Wild Wild Chat**.

It starts with **Bounty Hunting**, a quick scavenger hunt where two players watch the same stream chat and race to spot the right messages before the clock runs out.

:::media-right

![Bounty Hunting wanted board](./64a11deb8049b2c9.png){shadow=smooth;rotate=-6deg}

### How it works

Start a Playground match from live chat, invite another player, and wait a moment while the game prepares the round.

Each round gets six bounties based on things that naturally happen in chat. You might be asked to find a message with 3+ emojis, a message in all caps, a question, a user mention, a verified chatter, a link, a number, a repeated phrase, or one of the busier chatters.

Both players press **Ready**, then a short 3, 2, 1 countdown starts the real hunt. After that, you have 60 seconds.

:::


## Claiming bounties

The wanted board shows **Them vs You**, the live timer, and the six open bounties. Each bounty has a money value, a description, and an **Open** or **Claimed** stamp.

To claim one, click a live chat message. If the message matches an open bounty, the game stamps that bounty as claimed, adds the money to your score, and places your avatar on the row.

The first valid claim wins that bounty. Once it is claimed, it is closed for both players, so keep scanning chat for the next opening.


## Round over

The round ends when the timer hits zero or when all six bounties have been claimed.

After a short round-over screen, **The Ledger** shows the final result. The winner appears first, followed by the other player, with each player's avatar, rank, bounties claimed, and money earned. The player with the most money wins.


## Built for live chat

The Wild Wild Chat is only available during live chat, because the game is about reacting to the stream chat as it happens.

There is also a compact mode for this game. If the full wanted poster is covering too much chat, shrink the panel down to a small row that keeps the timer and score visible while leaving the chat easier to read.


## Part of Playground

Like Chess and HELP-A-FRIEND! Trivia, The Wild Wild Chat lives inside Playground. It uses the same Games panel, the same invite flow, and the same floating game window, so it stays close to YouTube chat instead of becoming a separate app.

The style is new, though: wanted posters, stamped claims, dusty paper, and a little western pizzazz.
