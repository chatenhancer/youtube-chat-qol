The next Playground game is riding into live chat: **The Wild Wild Chat**.

It starts with **Chat Bounties**, a quick scavenger hunt where two players watch the same stream chat and race to spot the right messages before the clock runs out.

:::media-right

![Chat Bounties wanted board](./64a11deb8049b2c9.png){shadow=smooth;rotate=-6deg}

### How it works

Start a Playground match from live chat, invite another player, and get ready for a 60-second round.

Each round chooses a fresh set of bounties from things that naturally happen in chat: a message with 3+ emojis, a message in all caps, a verified chatter, a question, a user mention, a new chatter, or a phrase that gets repeated.

When a bounty is claimed, it is stamped shut for both players. No duplicates, no second chances. Keep watching the chat because the next open bounty might appear immediately.

:::



## Part of Playground

Like Chess and HELP-A-FRIEND! Trivia, Chat Bounties lives inside Playground. It uses the same Games panel, the same invite flow, and the same compact floating game window, so it stays close to YouTube chat instead of becoming a separate app.

The style is new, though: wanted posters, stamped claims, dusty paper, and a little western pizzazz.





:::media-left

![Playground settings in the Chat Enhancer popup](./c99cee6a87fb4253.png)

Playground is still opt-in. Enable **Join Playground** from the extension settings, open a live stream with chat, and look for the Games button when the update arrives.

:::
