## Playground가 왔습니다

Playground는 Chat Enhancer 안에 있는 작은 게임 허브입니다. 같은 스트림에 있고 확장 프로그램을 설치한 다른 시청자와 함께 플레이할 수 있습니다.

:::media-right

![YouTube 라이브 채팅 위에 떠 있는 작은 체스 패널](./f779ebcc57f8280d.png){shadow=smooth rotation=-2}

게임은 작게 유지됩니다. 패널은 드래그할 수 있어서 채팅이 다시 바빠지면 방해되지 않는 곳으로 옮기면 됩니다.

:::

## 체스 플레이 방식

게임 패널을 열고 **체스**를 선택한 다음, 같은 스트림에서 참가 가능한 사람을 초대하세요. 상대가 수락하면 라이브 채팅 위의 작은 플로팅 패널에 보드가 열립니다.

게임은 체스의 일반 규칙을 따릅니다. 수는 전송 전에 확인되고, 턴은 두 플레이어 사이에서 동기화되며, 매치는 체크메이트, 무승부, 기권으로 끝날 수 있습니다. 스트림이 다시 바빠지면 패널을 옆으로 끌어 두고 계속 시청하면 됩니다.

주변에 다른 사람이 없다면 체스는 Computer 상대도 지원합니다. 플레이어 목록에서 **Computer (Beginner)**, **Computer (Club)** 또는 **Computer (Master)**를 선택하고, 다른 시청자와 하듯 매치를 시작하세요.

## 라이브 채팅에 어울리는 이유

Playground는 YouTube에 억지로 붙인 본격 게임방이 아닙니다. 채팅은 열려 있지만 크게 움직이지 않는, 스트림의 조용한 순간을 위한 기능입니다. 그래서 체스는 일부러 작게 만들었습니다.

- 컴팩트하고 이동 가능한 보드를 사용합니다.
- 현재 스트림에서 Chat Enhancer를 사용 중이고 참가 가능한 플레이어만 보여 줍니다.
- YouTube의 나머지 화면을 계속 보이게 하므로 바로 채팅으로 돌아갈 수 있습니다.

:::media-left

![Chat Enhancer 팝업의 Playground 설정](./c99cee6a87fb4253.png)

채팅에 게임 아이콘을 표시하려면 **Playground 참여**를 켜세요.

게임 패널 안에서 다른 플레이어에게 보이고 싶을 때 **게임 가능**을 켜세요. 보통 항상 참가 가능하게 두고 싶다면 확장 프로그램 설정에서 **기본적으로 가능**을 켜세요.

:::
