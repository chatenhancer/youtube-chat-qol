## Playground est là !

Playground est un petit hub de jeux dans Chat Enhancer. Il vous permet de jouer avec d’autres spectateurs qui ont installé l’extension et se trouvent dans le même live.

Le streamer s’absente une minute ? Ouvrez Playground, lancez une partie rapide et gardez le chat à portée de main.

:::media-right

![Panneau d’échecs compact flottant au-dessus du chat en direct YouTube](./f779ebcc57f8280d.png){shadow=smooth rotation=-2}

Les jeux restent compacts. Le panneau est déplaçable, vous pouvez donc le mettre de côté quand le chat reprend.

Les échecs ont été le premier jeu de Playground, et ils expliquent toujours le mieux l’idée : une petite activité rapide qui reste près du chat au lieu de prendre toute la page.

:::

## Comment fonctionnent les échecs

Ouvrez le panneau Jeux, choisissez **Échecs** et invitez une personne disponible dans le même live. Quand elle accepte, l’échiquier s’ouvre dans un petit panneau flottant au-dessus du chat en direct.

Le jeu utilise les règles normales des échecs. Les coups sont vérifiés avant d’être envoyés, les tours restent synchronisés entre les deux joueurs, et la partie peut se terminer par échec et mat, nulle ou abandon. Si le live s’anime à nouveau, faites glisser le panneau de côté et continuez à regarder.

Si personne d’autre n’est là, les échecs prennent aussi en charge les adversaires **Computer**. Choisissez **Computer (Beginner)**, **Computer (Club)** ou **Computer (Master)** dans la liste des joueurs et lancez une partie comme avec un autre spectateur.

## Pourquoi cela a sa place dans le chat en direct

Playground n’est pas une salle de jeux complète ajoutée à YouTube. Il existe pour les moments calmes d’un live, quand le chat reste ouvert mais qu’il ne se passe pas grand-chose.

C’est pourquoi les échecs restent volontairement petits :

- Ils se lancent depuis le panneau Jeux de l’extension.
- Ils utilisent un échiquier compact et déplaçable.
- Ils n’affichent que les joueurs disponibles qui utilisent aussi Chat Enhancer dans le live actuel.
- Ils gardent le reste de YouTube visible, pour revenir immédiatement au chat.

Les paramètres de Playground se trouvent dans le popup de l’extension.

:::media-left

![Paramètres Playground dans le popup Chat Enhancer](./c99cee6a87fb4253.png)

Activez **Rejoindre Playground** pour faire apparaître l’icône Jeux dans le chat.

Dans le panneau Jeux, activez **Disponible pour jouer** quand vous voulez que les autres joueurs vous voient. Si vous voulez généralement être disponible, activez **Disponible par défaut** dans les paramètres de l’extension.

:::

## Plus seulement les échecs

Playground a grandi depuis cette première présentation des échecs. Vous pouvez aussi jouer à [HELP-A-FRIEND! Trivia](/fr/blog/new-in-0-14-0-help-a-friend-trivia/), et [The Wild Wild Chat](/fr/blog/the-wild-wild-chat-coming-to-chat-enhancer-0-15-0/) transforme le chat en direct en une chasse aux primes rapide.

Le déroulement reste le même : ouvrez le panneau Jeux, invitez un spectateur ou un adversaire Computer quand c’est pris en charge, jouez une partie compacte et gardez le live à proximité.
