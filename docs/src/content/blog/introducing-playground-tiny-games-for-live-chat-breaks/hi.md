## Playground आ गया है!

Playground, Chat Enhancer के अंदर छोटा games hub है. यह आपको उन दूसरे viewers के साथ खेलने देता है जिनके पास extension installed है और जो उसी stream में हैं.

Streamer एक मिनट के लिए दूर हैं? Playground खोलें, जल्दी-सी match खेलें, और chat को पास रखें.

:::media-right

![YouTube live chat के ऊपर floating compact chess game panel](./f779ebcc57f8280d.png){shadow=smooth rotation=-2}

Games compact रहते हैं. Panel draggable है, इसलिए chat फिर तेज़ होने पर आप इसे रास्ते से हटा सकते हैं.

शतरंज पहला Playground game था, और यह idea अब भी सबसे अच्छी तरह समझाता है: page पर कब्ज़ा किए बिना chat के पास रहने वाली quick side activity.

:::

## शतरंज कैसे काम करता है

गेम panel खोलें, **शतरंज** चुनें, और उसी stream में available किसी व्यक्ति को invite करें. जब वह accept करेगा, board live chat के ऊपर एक छोटे floating panel में खुल जाएगा.

Game normal chess rules इस्तेमाल करता है. Moves भेजने से पहले check होते हैं, turns दोनों players के बीच synced रहते हैं, और match checkmate, draw या resignation से खत्म हो सकता है. अगर stream फिर busy हो जाए, panel को side में drag करें और देखते रहें.

अगर आसपास कोई और नहीं है, तो शतरंज **Computer** opponents को भी support करता है. Players list से **Computer (Beginner)**, **Computer (Club)** या **Computer (Master)** चुनें और match वैसे ही शुरू करें जैसे किसी दूसरे viewer के साथ करते.

## यह live chat में क्यों फिट बैठता है

Playground कोई पूरा game room नहीं है जिसे YouTube पर जोड़ दिया गया हो. यह stream के धीमे हिस्सों के लिए है, जब chat खुला रहता है लेकिन बहुत कुछ नहीं हो रहा होता.

इसलिए शतरंज जानबूझकर छोटा है:

- यह extension के अंदर गेम panel से चलता है.
- यह compact, movable board इस्तेमाल करता है.
- यह सिर्फ़ उन available players को दिखाता है जो current stream में Chat Enhancer भी इस्तेमाल कर रहे हैं.
- यह बाकी YouTube को visible रखता है, ताकि आप तुरंत chat में वापस आ सकें.

Playground settings extension popup में हैं.

:::media-left

![Chat Enhancer popup में Playground settings](./c99cee6a87fb4253.png)

Chat में गेम icon दिखाने के लिए **Playground से जुड़ें** enable करें.

गेम panel के अंदर, जब आप चाहते हैं कि दूसरे players आपको देखें, तो **खेलने के लिए उपलब्ध** on करें. अगर आप आम तौर पर available रहना चाहते हैं, तो extension settings में **डिफ़ॉल्ट रूप से उपलब्ध** on करें.

:::
