## Playground आ गया है!

Playground extension के अंदर रहने वाला games hub है, जहाँ आप उन users के साथ खेल सकते हैं जिनके पास भी extension installed है.

क्या आपका streamer break पर है? उनके वापस आने तक Playground खोल लें.

:::media-right

![YouTube live chat के ऊपर floating compact chess game panel](./f779ebcc57f8280d.png){shadow=smooth rotation=-2}

Games compact हैं और इससे बहुत बड़े नहीं होंगे. Panel draggable है, इसलिए आप इसे chat में कहीं भी रख सकते हैं.

पहला available game अच्छा पुराना chess है. आप अपने नए stream buddy को कितनी जल्दी checkmate कर सकते हैं?

:::

Extension settings के इस section में Playground options देखें.

:::media-left

![Chat Enhancer popup में Playground settings](./c99cee6a87fb4253.png)

Chat में Games icon दिखाने के लिए “Join Playground” setting enable करें.

जब आप Games panel खोलेंगे, तो “Available for games” option enable करना होगा ताकि दूसरे players आपको देख सकें, या extension settings में “Available by default” enable कर दें ताकि यह हमेशा on रहे. आपकी मर्ज़ी.

:::
