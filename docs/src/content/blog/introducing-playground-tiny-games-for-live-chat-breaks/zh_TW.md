## Playground 來了！

Playground 是擴充功能裡的遊戲中心，你可以和同樣安裝擴充功能的其他使用者一起玩遊戲。

實況主暫時離開了嗎？打開 Playground 等他回來。

:::media-right

![漂浮在 YouTube 直播聊天室上的精簡西洋棋遊戲面板](./f779ebcc57f8280d.png){shadow=smooth rotation=-2}

遊戲很精簡，不會比這個大太多。面板可以拖曳，所以你可以把它放在聊天室中的任何位置。

第一個可玩的遊戲是經典西洋棋。你能多快將死新的實況夥伴？

:::

在擴充功能設定的這個區段尋找 Playground 選項。

:::media-left

![Chat Enhancer popup 中的 Playground 設定](./c99cee6a87fb4253.png)

啟用「加入 Playground」設定後，遊戲圖示就會出現在聊天室中。

開啟遊戲面板時，你需要啟用「可參與遊戲」，讓其他玩家看得到你；也可以在擴充功能設定中啟用「預設可參與」，讓它一直開啟。由你決定。

:::

更多遊戲即將推出！
