## Playground 來了！

Playground 是 Chat Enhancer 裡的小型遊戲中心。你可以和同樣安裝擴充功能、並且在同一場直播裡的其他觀眾一起玩。

實況主暫時離開一分鐘？打開 Playground，快速玩一局，同時讓聊天室保持在旁邊。

:::media-right

![漂浮在 YouTube 直播聊天室上的精簡西洋棋遊戲面板](./f779ebcc57f8280d.png){shadow=smooth rotation=-2}

遊戲保持精簡。面板可以拖曳，所以聊天室又熱鬧起來時，你可以把它移到旁邊。

西洋棋是 Playground 的第一款遊戲，也仍然最能說明這個想法：靠近聊天室的快速小活動，而不是接管整個頁面。

:::

## 西洋棋如何運作

打開遊戲面板，選擇 **西洋棋**，然後邀請同一場直播中可參與的人。對方接受後，棋盤會在直播聊天室上方的小型浮動面板中打開。

遊戲使用一般西洋棋規則。走棋會在送出前檢查，雙方回合同步，棋局可以用將死、和棋或認輸結束。如果直播又忙起來，把面板拖到旁邊繼續觀看即可。

如果周圍沒有其他人，西洋棋也支援 **Computer** 對手。你可以從玩家清單中選擇 **Computer (Beginner)**、**Computer (Club)** 或 **Computer (Master)**，像邀請其他觀眾一樣開始一局。

## 為什麼它屬於直播聊天

Playground 不是硬塞進 YouTube 的完整遊戲房間。它是為直播中較慢的片段準備的：聊天室還開著，但沒有太多事情發生。

所以西洋棋有意保持小巧：

- 它從擴充功能內的遊戲面板執行。
- 它使用精簡、可移動的棋盤。
- 它只顯示目前直播中同樣使用 Chat Enhancer 的可參與玩家。
- 它讓 YouTube 的其他部分保持可見，讓你可以立刻回到聊天室。

Playground 設定位於擴充功能 popup 中。

:::media-left

![Chat Enhancer popup 中的 Playground 設定](./c99cee6a87fb4253.png)

啟用 **加入 Playground** 後，遊戲圖示就會出現在聊天室中。

在遊戲面板中，當你希望其他玩家看到你時，開啟 **可以一起玩**。如果你通常都想保持可參與，請在擴充功能設定中開啟 **預設顯示為可玩**。

:::

## 現在不只有西洋棋

自從這篇最初的西洋棋預覽之後，Playground 已經繼續擴展。你也可以玩 [HELP-A-FRIEND! Trivia](/zh-TW/blog/new-in-0-14-0-help-a-friend-trivia/)，而 [The Wild Wild Chat](/zh-TW/blog/the-wild-wild-chat-coming-to-chat-enhancer-0-15-0/) 會把直播聊天變成一場快速賞金狩獵。

流程保持不變：打開遊戲面板，在支援時邀請觀眾或 Computer 對手，玩一局精簡的遊戲，並讓直播保持在旁邊。
