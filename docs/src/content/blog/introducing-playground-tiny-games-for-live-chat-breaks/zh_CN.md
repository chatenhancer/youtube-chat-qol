## Playground 登场

Playground 是 Chat Enhancer 里的小型游戏中心。你可以和同样安装了扩展、并且在同一场直播里的其他观众一起玩。

:::media-right

![漂浮在 YouTube 直播聊天上方的紧凑的国际象棋游戏面板](./f779ebcc57f8280d.png){shadow=smooth rotation=-2}

游戏面板保持小巧，可以拖动。聊天重新热闹起来时，你可以把它挪到一边。

:::

## 国际象棋如何运作

打开游戏面板，选择 **国际象棋**，然后邀请同一场直播里可邀请的观众。对方接受后，棋盘会在直播聊天上方的小型浮动面板中打开。

游戏使用标准国际象棋规则。走棋会在发送前检查，双方回合同步，棋局可以通过将死、和棋或认输结束。如果直播又忙起来，把面板拖到一边继续看就行。

如果周围没有其他人，国际象棋也支持 Computer 对手。你可以从玩家列表中选择 **Computer (Beginner)**、**Computer (Club)** 或 **Computer (Master)**，像邀请其他观众一样开始一局。

## 为什么它属于直播聊天

Playground 不是硬塞进 YouTube 的完整游戏房间。它是为直播里节奏放慢的时刻准备的：聊天还开着，但暂时没太多事情发生。 所以国际象棋有意保持小巧：

- 它使用小巧、可拖动的棋盘。
- 它只显示当前直播里同样使用 Chat Enhancer 的可用玩家。
- 它让 YouTube 其他部分保持可见，方便你随时回到聊天。

:::media-left

![Chat Enhancer 弹窗中的 Playground 设置](./c99cee6a87fb4253.png)

启用 **加入 Playground** 后，聊天中会显示 游戏 图标。

在游戏面板里，想让其他玩家看到你时，开启 **可接收邀请**。如果你通常都愿意被邀请，在扩展设置中开启 **默认可接收邀请**。

:::

## 现在不只有国际象棋

自从这篇最初的国际象棋预览之后，Playground 已经继续扩展。你也可以玩 [HELP-A-FRIEND! Trivia](/zh-CN/blog/new-in-0-14-0-help-a-friend-trivia/)，而 [The Wild Wild Chat](/zh-CN/blog/the-wild-wild-chat-coming-to-chat-enhancer-0-15-0/) 会把直播聊天变成一场快节奏的 Bounty Hunting。
