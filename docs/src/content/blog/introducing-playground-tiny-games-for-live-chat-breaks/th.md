## Playground มาแล้ว!

Playground คือศูนย์รวมเกมที่อยู่ในส่วนขยาย ให้คุณเล่นกับผู้ใช้อื่นที่ติดตั้งส่วนขยายเหมือนกัน

สตรีมเมอร์ของคุณพักอยู่หรือเปล่า? เปิด Playground เล่นจนกว่าเขาจะกลับมาได้เลย

:::media-right

![แผงหมากรุกขนาดกะทัดรัดลอยอยู่เหนือ YouTube live chat](./f779ebcc57f8280d.png){shadow=smooth rotation=-2}

เกมมีขนาดกะทัดรัดและจะไม่ใหญ่ไปกว่านี้มากนัก แผงสามารถลากได้ คุณจึงวางไว้ตรงไหนก็ได้ใน chat

เกมแรกที่มีคือหมากรุกแบบคลาสสิก คุณจะ checkmate เพื่อนใหม่ใน stream ได้เร็วแค่ไหน?

:::

มองหาตัวเลือก Playground ในส่วนนี้ของการตั้งค่าส่วนขยาย

:::media-left

![การตั้งค่า Playground ใน popup ของ Chat Enhancer](./c99cee6a87fb4253.png)

เปิดการตั้งค่า “เข้าร่วม Playground” เพื่อให้ไอคอนเกมปรากฏใน chat

เมื่อเปิดแผงเกม คุณต้องเปิด “พร้อมสำหรับเกม” เพื่อให้ผู้เล่นคนอื่นเห็นคุณ หรือเปิด “พร้อมโดยค่าเริ่มต้น” ในการตั้งค่าส่วนขยายเพื่อให้เปิดไว้เสมอ แล้วแต่คุณเลือก

:::
