## Playground มาแล้ว!

Playground คือศูนย์รวมเกมขนาดเล็กใน Chat Enhancer ให้คุณเล่นกับผู้ชมคนอื่นที่ติดตั้งส่วนขยายและอยู่ในสตรีมเดียวกัน

สตรีมเมอร์หายไปสักนาทีหรือเปล่า? เปิด Playground เล่นแมตช์สั้น ๆ แล้วเก็บ chat ไว้ใกล้ตัว

:::media-right

![แผงหมากรุกขนาดกะทัดรัดลอยอยู่เหนือ YouTube live chat](./f779ebcc57f8280d.png){shadow=smooth rotation=-2}

เกมยังคงกะทัดรัด แผงสามารถลากได้ คุณจึงย้ายออกไปด้านข้างเมื่อ chat กลับมาคึกคักอีกครั้ง

หมากรุกเป็นเกมแรกของ Playground และยังอธิบายไอเดียได้ดีที่สุด: กิจกรรมสั้น ๆ ข้าง chat ที่ไม่ยึดทั้งหน้า

:::

## หมากรุกทำงานอย่างไร

เปิดแผงเกม เลือก **หมากรุก** แล้วเชิญคนที่พร้อมในสตรีมเดียวกัน เมื่ออีกฝ่ายตอบรับ กระดานจะเปิดในแผงลอยขนาดเล็กเหนือ live chat

เกมใช้กติกาหมากรุกปกติ การเดินจะถูกตรวจสอบก่อนส่ง เทิร์นของผู้เล่นทั้งสองจะซิงก์กัน และแมตช์อาจจบด้วย checkmate เสมอ หรือยอมแพ้ ถ้าสตรีมกลับมายุ่งอีกครั้ง ให้ลากแผงไปด้านข้างแล้วดูต่อได้เลย

ถ้าไม่มีคนอื่นอยู่ใกล้ ๆ หมากรุกยังรองรับคู่แข่ง **Computer** ด้วย เลือก **Computer (Beginner)**, **Computer (Club)** หรือ **Computer (Master)** จากรายชื่อผู้เล่น แล้วเริ่มแมตช์เหมือนเล่นกับผู้ชมคนอื่น

## ทำไมจึงเหมาะกับ live chat

Playground ไม่ใช่ห้องเกมเต็มรูปแบบที่แปะอยู่บน YouTube แต่มีไว้สำหรับช่วงที่สตรีมเงียบลง เมื่อ chat ยังเปิดอยู่แต่ไม่ค่อยมีอะไรเกิดขึ้น

นั่นทำให้หมากรุกตั้งใจถูกทำให้เล็ก:

- ทำงานจากแผงเกมภายในส่วนขยาย
- ใช้กระดานขนาดกะทัดรัดที่ย้ายได้
- แสดงเฉพาะผู้เล่นที่พร้อมและใช้ Chat Enhancer ในสตรีมปัจจุบันเหมือนกัน
- ทำให้ส่วนอื่นของ YouTube ยังมองเห็นได้ เพื่อให้คุณกลับเข้า chat ได้ทันที

การตั้งค่า Playground อยู่ใน popup ของส่วนขยาย

:::media-left

![การตั้งค่า Playground ใน popup ของ Chat Enhancer](./c99cee6a87fb4253.png)

เปิด **เข้าร่วม Playground** เพื่อให้ไอคอนเกมปรากฏใน chat

ในแผงเกม ให้เปิด **พร้อมเล่น** เมื่อคุณต้องการให้ผู้เล่นคนอื่นเห็นคุณ ถ้าปกติอยากพร้อมเล่นอยู่แล้ว ให้เปิด **พร้อมเล่นเป็นค่าเริ่มต้น** ในการตั้งค่าส่วนขยาย

:::

## ตอนนี้ไม่ได้มีแค่หมากรุก

Playground เติบโตขึ้นตั้งแต่ตัวอย่างหมากรุกแรกนี้ คุณยังเล่น [HELP-A-FRIEND! Trivia](/th/blog/new-in-0-14-0-help-a-friend-trivia/) ได้ด้วย และ [The Wild Wild Chat](/th/blog/the-wild-wild-chat-coming-to-chat-enhancer-0-15-0/) จะเปลี่ยน live chat ให้เป็นเกมล่าค่าหัวที่รวดเร็ว

ขั้นตอนยังเหมือนเดิม: เปิดแผงเกม เชิญผู้ชมหรือคู่แข่ง Computer เมื่อรองรับ เล่นแมตช์ขนาดกะทัดรัด และเก็บสตรีมไว้ใกล้ตัว
