Das zweite Playground-Spiel ist da: **HELP-A-FRIEND! Trivia**

:::media-right

![HELP-A-FRIEND! Trivia-Logo](./2026-06-12-new-in-0-14-0-help-a-friend-trivia-assets/6ef6d6f2584f127f.png){shadow=smooth;rotate=-8deg}

Statt auf einem Quizbrett spielt sich *HELP-A-FRIEND! Trivia* wie ein kleiner Gruppenchat. Einer deiner Freunde hat beim Stream offensichtlich nicht aufgepasst und braucht jetzt Hilfe... hast du das Wissen?

Richtige Antworten bekommen die 🏆-Reaktion.

Falsche Antworten werden *höflich* beurteilt.

:::

## So funktioniert es

Starte ein Playground-Match aus einem YouTube-Replay (einem beendeten Livestream), lade einen weiteren Spieler ein und warte ein paar Sekunden, während die Fragen vorbereitet werden.

Sobald das Spiel beginnt, fragt dein "Freund" nach dem Replay, vier mögliche Antworten erscheinen, und beide Spieler wählen, bevor die Zeit abläuft. Antworte schnell. Dein Kumpel ist nicht geduldig.

## Für Replays gebaut

Jedes Match wird aus dem Transkript des Replays generiert, das du gerade ansiehst. So kann das Spiel nach Momenten fragen, die in diesem Stream wirklich passiert sind: Enthüllungen, Auszeichnungen, Witze, Abschweifungen und alles andere, was es ins Video geschafft hat.

:::media-left

![HELP-A-FRIEND! Trivia-Screenshot](./f6e02e23a9bb8118.png)

## Probier es aus!

*HELP-A-FRIEND! Trivia* ist Teil von Playground, das weiterhin optional ist. Aktiviere Playground in den Erweiterungseinstellungen, öffne ein Replay mit Livechat und starte ein Match im Spiele-Panel. Achte im Chat auf das Controller-Symbol.

Aktuell auf Englisch verfügbar.

:::
