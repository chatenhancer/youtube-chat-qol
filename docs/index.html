Privacy

Local-first chat tools

The extension is designed to keep routine chat enhancements on your device. Translation is the one feature that needs an external service, and it only runs after you choose a target language.

Read the full privacy policy

Most features run in your browser. Settings are saved with your browser's extension storage so they can follow your extension install. Your mentions inbox and frequent emoji counts are stored locally on your device.

Recent profile messages are only kept in memory for the current chat page. When you leave or reload the page, that temporary recent-message context is cleared.

Translation is off by default. When you choose a translation language, chat message text is sent to the Google Translate endpoint so the translated text can be shown in chat.

Chat Enhancer for YouTube does not sell user data and is not affiliated with YouTube or Google.

Reference

Chat commands expand with Tab

Type a command in YouTube's chat input, then press Tab. Commands do not send automatically, and unknown slash commands are left alone. Use // to send a literal slash command.

Inline commands: /mention, /reply, /time, and /timeuntil can expand inside a sentence.

Whole-input commands: /help, /quote, /again, /repeat, and all /set... commands should be the only text in the input.

Text Commands

/mention or /reply
Insert a mention for the author of your newest mentions-inbox message.
/quote
Quote the newest message saved in your mentions inbox.
/again or /repeat
Restore your last sent message, including rich emoji content when available.
/time utc
Insert the current time for a supported timezone or city.
/timeuntil 7:45pm
Insert the time remaining until the next matching local time.
/help
Open the command help card inside YouTube chat.

Time aliases include utc, tokyo, jst, seoul, kst, london, paris, madrid, nyc, et, la, and pt.

Setting Commands

/settranslateto english
Set the translation target language. Use off to disable translation.
/settranslationdisplay replace
Replace original messages with translations. Use below to show translations underneath.
/setquotelength 120
Set quote length to 80, 120, 180, or 240.
/setmentionsound on
Turn mention sounds on or off.
/setopenchannelsinpopup on
Open channels in popup windows. Use off to open them normally.

/timeuntil accepts formats like 7, 7am, 7:45 am, 19:00, and 19:00:30.
