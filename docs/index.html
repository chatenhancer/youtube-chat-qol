Built for live streams

Small tools where chat already happens

Translate live chat

Choose a target language in the extension popup, then quickly toggle translation from YouTube's live chat settings.

Mention and quote

Use YouTube's message menu, author clicks, profile cards, or slash commands to mention and quote people faster.

Chat context

Open an avatar card to review a user's recent messages in the current chat and jump to their channel when needed.

Inbox

Keep a local list of messages that mention your handle or match your keywords, with an optional subtle sound for new inbox messages.

Frequent emojis

Add a compact row of your most-used emojis to YouTube's emoji picker so common reactions are always close.

Chat commands

Press Tab to expand commands for mentions, quotes, repeated messages, time helpers, and extension settings.

Privacy

Local-first chat tools

The extension is designed to keep routine chat enhancements on your device. Translation is the one feature that needs an external service, and it only runs after you choose a target language.

Read the full privacy policy
Scope
Runs only on YouTube live chat and live chat replay pages.
Storage
Settings, inbox messages, watched keywords, and frequent emoji counts stay in browser extension storage.
Translation
Off by default. If enabled, message text is sent to Google Translate without your YouTube cookies or credentials.
Tracking
No analytics, browsing history collection, data resale, or extension-owned server.

Reference

Chat commands expand with Tab

Type a command in YouTube's chat input, then press Tab. Commands do not send automatically, and unknown slash commands are left alone. Use // to send a literal slash command.

Inline commands: /mention, /reply, /time, and /timeuntil can expand inside a sentence.

Whole-input commands: /quote, /again, /repeat, /help, and all /set... commands should be the only text in the input.

Text commands

/mention or /reply
Insert a mention for the author of your newest inbox message.
/quote
Quote the newest message saved in your inbox.
/again or /repeat
Restore your last sent message, including rich emoji content when available.
/time utc
Insert the current time for a supported timezone or city.
Supported aliases: utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/timeuntil 7:45pm
Insert the time remaining until the next matching local time.
Accepted formats: 7, 7am, 7 am, 7:45am, 7:45 am, 19, 19:00, 19:00:30, and 7:45:30 pm.
/help
Open the command help card inside YouTube chat.

Setting commands

/settranslateto english or /settranslateto off
Set the translation target language. Use off to disable translation.
/settranslationdisplay replace or /settranslationdisplay below
Replace original messages with translations. Use below to show translations underneath.
/setquotelength 120
Set quote length to 80, 120, 180, or 240.
/setsound on or /setsound off
Turn inbox sounds on or off.
/setopenchannelsinpopup on or /setopenchannelsinpopup off
Open channels in popup windows. Use off to open them normally.
