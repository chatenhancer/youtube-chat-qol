Getting started

Install, refresh YouTube once if it was already open, then use the extension where live chat already is.

  1. Install from your browser store Use the Chrome or Firefox button above. Other Chromium-based browsers can generally use the Chrome build.
  2. Open or refresh YouTube live chat Refresh any YouTube live chat tabs that were already open. A brief startup effect may play; you'll know the chat enhancements are ready when the inbox icon appears in YouTube chat.
  3. Adjust full settings when needed The extension works in live chat automatically. For complete settings, click the browser toolbar extensions icon , and choose Chat Enhancer to reveal all the options. Pin it if you want faster access.
  4. Use the chat enhancements Message menus, avatar cards, the inbox button, frequent emojis, and Tab commands appear and can be used within YouTube's chat interface.

Privacy

Local-first chat tools

The extension is designed to keep routine chat enhancements on your device. Translation is the one feature that needs an external service, and it only runs after you choose a target language.

Read the full privacy policy
Scope
Runs only on YouTube live chat and live chat replay pages.
Storage
Settings, inbox messages, watched keywords, and frequent emoji counts stay in browser extension storage.
Translation
Off by default. If enabled, incoming chat text or draft text you choose to translate is sent to Google Translate without your YouTube cookies or credentials.
Tracking
No analytics, browsing history collection, data resale, or extension-owned server.

Reference

Chat commands expand with Tab

Type a command in YouTube's chat box, then press Tab. Commands do not send automatically, and unknown slash commands are left alone. Use // to send a literal slash command.

Inline commands: /mention, /m, /reply, /r, /time, /t, /when, /wh, /timeuntil, /tu, /timesince, /ts, /translate, and /tr can expand inside a sentence.

Whole-input commands: /quote, /q, /again, /repeat, /watch, /w, /unwatch, /uw, /focus, /f, /who, /help, and all /set... commands should be the only text in the input.

Text commands

/mention Aliases: /m /reply /r
Insert a mention for the author of your newest inbox message.
/quote Aliases: /q
Without a handle, quote your newest inbox message. With @handle, quote that user's latest seen message.
/again Aliases: /repeat
Restore your last sent message, including rich emoji content when available.
/focus @handle Aliases: /f
With @handle, open focus mode for that user. Without a handle, focus the latest user who mentioned you.
/who @handle
Open a recent user card.
/help
Open the command help card inside YouTube chat.

Inbox

/watch keyword Aliases: /w
Add watched keywords or phrases. Without text, show your current watch list.
/unwatch keyword Aliases: /uw
Stop watching keywords or phrases.

Time and translation

/time Aliases: /t Example: /time utc
Insert the current local time, or the current time for a supported place/timezone.
Supported aliases: utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/when 2026-05-29 8pm pt Aliases: /wh /timeuntil /tu /timesince /ts
Insert the duration until or since a time/date, optionally in a supported place/timezone.
Accepted formats: 2026-05-29 8pm pt, 2026-06-12 19:00 london, 8pm 2026-06-12 london, tokyo 7pm, and 7pm tokyo.
Supported places/timezones: utc, tokyo, jst, seoul, kst, london, paris, madrid, newyork, nyc, et, eastern, losangeles, la, pt, and pacific.
/translate es hello Aliases: /tr
Translate custom text in the chat box before sending it.

Setting commands

/settranslateto english Aliases: /lang en /lang off
Set the translation language. Use off to disable translation. Use /lang without a language to show the current target.
/settranslationdisplay replace|below
Replace original messages with translations. Use below to show translations underneath.
/setsound on|off
Turn inbox sounds on or off.
